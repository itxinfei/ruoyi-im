# 16 设计复刻差异

> 用途：把当前 UI/页面与钉钉 Windows 客户端对齐所需的差异点写清，作为大模型/设计师复刻参考。  
> 说明：以 `21-IM桌面端页面规范.md` 作为基准样式说明，结合实际页面输出差异并给出落地方向。

## 1. 使用规则

1. 每次调整样式前，先对照本表确认属于哪组差异。
2. 明显偏离钉钉界面时，不直接动手补功能，而是先更新差异表及任务卡 `notes`。
3. 设计复刻任务必须附带视觉对比（稿、截图），并在 `llm-tasks.yaml` 中引用本文件。

## 3. 功能缺口与设计不完善点

> **更新时间**：2026-04-01

### 3.1 已解决项 ✅

1. ~~组件样式与钉钉底层变量（色彩、线宽、间距）未统一~~ → 已补充完整 CSS 变量规范
2. ~~暗色模式缺少完整色板~~ → 已补充 `[data-theme="dark"]` 完整色板
3. ~~过渡动画时长不规范~~ → 已定义 100ms/200ms/250ms/300ms 四档标准
4. ~~阴影层级规范缺失~~ → 已定义 S1/S2/S3 三级阴影
5. ~~消息发送状态机 UI 描述不完整~~ → 已补充完整状态机规范
6. ~~消息操作工具条定位不明确~~ → 已明确定位规范

### 3.2 待解决项 ⏳

1. 多个页面存在空白卡或"开发中"提示，需在对应任务卡中标记为 `unknown` 并补上真实入口。
2. 弹窗/详情/工具栏之间间距不一致，满足钉钉标准需要：
   - 固定高度与边距（顶部 16px、左右 20px）
   - 按键按钮在 36x36 内，不超过 300ms 圆角风格
3. 交互反馈（悬停、选中、禁止）缺少钉钉式渐变或阴影，需补 state 颜色规范。
4. 多媒体消息气泡（图片/视频/语音）尺寸规范待补充。
