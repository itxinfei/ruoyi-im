# 37 云盘与文档模块UI设计规范

> **用途**：定义云盘与协同文档模块的页面布局、文件列表规范及交互标准。  
> **设计基准**：钉钉 Windows 8.2.0 桌面端，遵循 `docs/27-PC版钉钉UI设计蓝图.md`。

---

## 1. 页面定位与模板

*   **模块 ID**：`documents`
*   **页面模板**：列表-详情协同页 (`L1 + L2 + L3 + Toolbar`)
*   **主要任务**：
    1.  管理个人与部门文件。
    2.  文件上传、下载与分享。
    3.  在线预览（图片、视频、PDF、Office）。
    4.  多级文件夹管理。

---

## 2. 空间布局 (Layout)

### 2.1 整体骨架
*   **L1 (一级导航)**：深色侧边栏，宽度 `64px`。
*   **L2 (目录侧边栏)**：宽度 `240px`，承载固定筛选（全部、收藏、最近、回收站）和文件夹树。
*   **L3 (主列表区)**：`flex: 1`，承载文件列表或网格视图。
*   **Toolbar (操作栏)**：位于 L3 顶部，高度 `48px`，含“上传”、“新建”、“视图切换”、“搜索”。

### 2.2 列表与网格切换
*   **列表视图**：适合大量文件的管理，显示文件图标、名称、大小、修改时间。
*   **网格视图**：适合图片/视频预览，卡片尺寸 `120px * 160px`，大图居中。

---

## 3. 组件级设计规范

### 3.1 文件图标规范 (File Icons)
*   严格按照 `design-tokens.scss` 中的颜色定义：
    *   PDF: `--dt-file-pdf` (#FF4D4F)
    *   Word: `--dt-file-word` (#2B7DFF)
    *   Excel: `--dt-file-excel` (#22AB5C)
    *   PPT: `--dt-file-ppt` (#FF7A00)
    *   Other: `--dt-file-other` (#ADB1B8)

### 3.2 列表行 (File Row)
*   **高度**：`48px`。
*   **交互**：
    *   Hover：整行变色 `var(--dt-bg-hover)`，右侧显示“分享”、“收藏”、“删除”快捷操作。
    *   选中：背景色 `#E6F2FF`，左侧出现复选框。

### 3.3 传输管理器 (Transfer Manager)
*   **形态**：右下角悬浮窗或独立 Tab。
*   **状态**：
    *   上传中：显示进度条、速度、剩余时间。
    *   失败：显示红色警告图标，支持“重试”。
    *   完成：自动消失或显示提示。

---

## 4. 状态清单 (States)

| 状态 | 表现 |
|-----|------|
| **Uploading** | 列表行出现微弱的背景扫描动效，进度百分比实时更新。 |
| **Empty** | 空文件夹显示“将文件拖拽至此上传”，支持点击中间的大图标触发上传。 |
| **Previewing** | 全屏蒙层（Overlay），支持缩放、旋转、下载。 |
| **Drag & Drop** | 拖入文件时，主区边缘出现蓝色虚线边框，显示“松开上传”。 |

---

## 5. 交互路径 (Interactions)

1.  **上传文件**：点击“上传” -> 弹出系统文件选择器 -> 选中后右下角弹出进度框。
2.  **右键菜单**：右键文件 -> 弹出菜单（预览、下载、分享、重命名、移动、删除）。
3.  **面包屑导航**：点击“全部文件 > 项目文档”中的“全部文件” -> 列表回退到根目录。

---

## 6. 验收点 (AC)

- [ ] AC1: 支持大文件（>100MB）的分片上传进度实时反馈。
- [ ] AC2: 右键菜单在列表底部时自动向上弹出，防止溢出屏幕。
- [ ] AC3: 文件夹双击进入，文件单击选中/双击预览。
- [ ] AC4: 列表加载支持触底自动分页（Infinite Scroll）。
