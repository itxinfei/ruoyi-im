## 二、后端实现 (JDK 1.8 规范)

### 2.1 日程基础逻辑

```java
@Service
@RequiredArgsConstructor
public class ImScheduleServiceImpl implements ImScheduleService {

    private final ImScheduleMapper scheduleMapper;

    /**
     * 创建日程
     * 遵循 JDK 1.8，严禁 var
     */
    @Transactional(rollbackFor = Exception.class)
    public void createSchedule(ScheduleDTO scheduleDTO) {
        ImSchedule schedule = new ImSchedule();
        // 显式 DTO -> Entity 转换
        BeanUtils.copyProperties(scheduleDTO, schedule);
        schedule.setCreateTime(new Date());
        scheduleMapper.insert(schedule);
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 日程列表封装

```javascript
/**
 * 日程管理逻辑封装 (纯 JS)
 */
import { ref } from 'vue';
import { scheduleApi } from '@/api/schedule';

export function useSchedule() {
  const schedules = ref([]);

  const loadSchedules = async (month) => {
    const result = await scheduleApi.listByMonth(month);
    schedules.value = result.data || [];
  };

  return {
    schedules,
    loadSchedules
  };
}
```

## 四、UI 表现与对齐规范 (DingTalk Style)

### 4.1 日历网格视图 (Calendar Grid)
- **基础色调**：背景色 `#FFFFFF`。星期表头文字 `12px`，色值 `rgba(23, 26, 29, 0.4)`。
- **今日高亮 (Today)**：当前日期的数字背景必须是一个 `24x24` 的正圆形，颜色为 `#0089FF`，文字为 `#FFFFFF`。
- **事件指示点 (Event Dots)**：某天有日程时，数字下方显示一个 `4x4` 的小圆点（默认为品牌蓝）。

### 4.2 侧边日程面板 (Schedule Panel)
- **布局**：日历组件右侧固定 `320px` 宽度的面板，展示选中日期的全部日程。
- **日程卡片**：
  - 左侧展示竖线（颜色代表日程优先级/类别）。
  - 主体文字 `14px`，时间 `12px`。Hover 出现编辑图标。
