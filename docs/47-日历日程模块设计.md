## 二、后端实现 (JDK 1.8 规范)

### 2.1 日程基础逻辑

```java
@Service
@RequiredArgsConstructor
public class ImScheduleServiceImpl implements ImScheduleService {

    private final ImScheduleMapper scheduleMapper;

    /**
     * 创建日程
     * 遵循 JDK 1.8，严禁 var
     */
    @Transactional(rollbackFor = Exception.class)
    public void createSchedule(ScheduleDTO scheduleDTO) {
        ImSchedule schedule = new ImSchedule();
        // 显式 DTO -> Entity 转换
        BeanUtils.copyProperties(scheduleDTO, schedule);
        schedule.setCreateTime(new Date());
        scheduleMapper.insert(schedule);
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 日程列表封装

```javascript
/**
 * 日程管理逻辑封装 (纯 JS)
 */
import { ref } from 'vue';
import { scheduleApi } from '@/api/schedule';

export function useSchedule() {
  const schedules = ref([]);

  const loadSchedules = async (month) => {
    const result = await scheduleApi.listByMonth(month);
    schedules.value = result.data || [];
  };

  return {
    schedules,
    loadSchedules
  };
}
```

## 四、UI 表现与对齐规范 (DingTalk Style)

### 4.1 日历整体布局 (Calendar Layout)

- **页面背景**：`var(--dt-bg-body) (#f5f6f7)`
- **顶部工具栏**：
  - 高度 `56px`，背景 `#FFFFFF`
  - 左侧：月份切换（`<` `>` 按钮 + 当前月份文字）
  - 右侧：视图切换按钮组 [月] [周] [日]

### 4.2 日历网格视图 (Calendar Grid)

#### 4.2.1 月视图 (Month View)

- **网格尺寸**：
  - 单元格宽度：自适应（`calc((100% - 6px) / 7)`）
  - 单元格高度：固定 `100px`
  - 单元格间距：`1px`，色值 `rgba(23, 26, 29, 0.08)`

- **星期表头**：
  - 高度 `40px`，背景 `#FFFFFF`
  - 文字 `12px`，色值 `rgba(23, 26, 29, 0.4)`
  - 文字居中，顺序：日、一、二、三、四、五、六

- **日期单元格**：
  - **日期数字**：`14px`，色值 `#171A1D`
  - **非本月日期**：色值 `rgba(23, 26, 29, 0.3)`
  - **今日高亮**：
    - 数字背景：`24x24` 正圆形，色值 `#277EFB`
    - 数字文字：`#FFFFFF`，粗体
  - **选中日期**：
    - 背景：`#E6F2FF`
    - 边框：`1px solid #277EFB`

- **事件指示点 (Event Dots)**：
  - 尺寸：`4x4` 圆点
  - 颜色：按优先级显示（高/中/低 对应 红/橙/蓝）
  - 位置：日期数字下方，最多显示 3 个
  - 间距：`2px`

- **事件条 (Event Bar)**：
  - 高度：`18px`
  - 圆角：`2px`
  - 左侧边框：`3px` 宽，按优先级着色
  - 文字：`12px`，单行截断
  - 最多显示：每单元格 2 条，超出显示 `+N`

#### 4.2.2 周视图 (Week View)

- **时间轴**：
  - 左侧固定宽度 `60px`
  - 时间刻度：每小时一行，高度 `60px`
  - 时间文字：`12px`，色值 `rgba(23, 26, 29, 0.4)`

- **日期列**：
  - 顶部日期行高度 `60px`
  - 当日列背景：`#E6F2FF`

- **事件块**：
  - 最小高度：`30px`（半小时）
  - 边框：`1px solid rgba(23, 26, 29, 0.08)`
  - 圆角：`4px`

#### 4.2.3 日视图 (Day View)

- **布局**：与周视图类似，仅显示单列
- **事件区域**：宽度更大，事件块显示更多信息

### 4.3 视图切换动画 (View Transition)

- **切换方式**：淡入淡出 + 轻微缩放
- **动画时长**：`250ms`
- **缓动函数**：`ease-out`
- **CSS 实现**：
```css
.calendar-view-enter-active,
.calendar-view-leave-active {
  transition: opacity 0.25s ease-out, transform 0.25s ease-out;
}
.calendar-view-enter-from {
  opacity: 0;
  transform: scale(0.95);
}
.calendar-view-leave-to {
  opacity: 0;
  transform: scale(1.05);
}
```

- **日程卡片 (Schedule Card)**：

  - 背景：`#FFFFFF`
  - 边框：`1px solid rgba(23, 26, 29, 0.08)`

| 优先级 | 竖线/圆点色 | 背景色（淡） | 说明 |

### 4.6 创建/编辑日程弹窗

- **尺寸**：宽度 `480px`
- **内边距**：`24px`

- **表单字段**：
  - 标题：输入框，高度 `36px`
  - 时间：日期时间选择器
  - 优先级：单选按钮组
  - 地点：输入框
  - 备注：文本域，高度 `80px`
  - 提醒：下拉选择（提前5分钟/15分钟/30分钟/1小时）
  - 重复：下拉选择（不重复/每天/每周/每月）

- **底部按钮**：
  - 取消：宽度 `80px`
  - 保存：背景 `#277EFB`，宽度 `80px`

### 4.7 日程提醒通知

- **桌面通知**：
  - 标题：日程名称
  - 内容：时间 + 地点
  - 图标：日历图标
  - 操作按钮：[查看详情] [稍后提醒]
