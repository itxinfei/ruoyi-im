## 二、后端实现 (JDK 1.8 规范)

### 2.1 日程基础逻辑

```java
@Service
@RequiredArgsConstructor
public class ImScheduleServiceImpl implements ImScheduleService {

    private final ImScheduleMapper scheduleMapper;

    /**
     * 创建日程
     * 遵循 JDK 1.8，严禁 var
     */
    @Transactional(rollbackFor = Exception.class)
    public void createSchedule(ScheduleDTO scheduleDTO) {
        ImSchedule schedule = new ImSchedule();
        // 显式 DTO -> Entity 转换
        BeanUtils.copyProperties(scheduleDTO, schedule);
        schedule.setCreateTime(new Date());
        scheduleMapper.insert(schedule);
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 日程列表封装

```javascript
/**
 * 日程管理逻辑封装 (纯 JS)
 */
import { ref } from 'vue';
import { scheduleApi } from '@/api/schedule';

export function useSchedule() {
  const schedules = ref([]);

  const loadSchedules = async (month) => {
    const result = await scheduleApi.listByMonth(month);
    schedules.value = result.data || [];
  };

  return {
    schedules,
    loadSchedules
  };
}
```
