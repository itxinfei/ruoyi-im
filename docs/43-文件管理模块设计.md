## 二、后端实现 (JDK 1.8 规范)

### 2.1 文件上传处理逻辑

```java
/**
 * 高级文件与二进制对象存储服务
 * 主要负责文件二进制流传输、MD5秒传及切片合成
 */
@Service
@RequiredArgsConstructor
public class ImFileServiceImpl implements ImFileService {

    /**
     * 执行简单的串行单文件上传逻辑
     * 强制遵循 DTO 传参与 VO 输出的隔离规范，及 JDK 1.8 生态
     *
     * @param file 由 Spring 容器包裹的表单多媒体文件对象
     * @return 含有前端所需的唯一凭证及对象 URL 的前端视图对象
     */
    @Transactional(rollbackFor = Exception.class)
    public FileVO upload(MultipartFile file) {
        if (Objects.isNull(file) || file.getSize() == 0) {
            throw new BusinessException(400, "文件不能为空");
        }
        
        String originalName = file.getOriginalFilename();
        // 存储逻辑实现...
        FileVO fileVO = new FileVO();
        fileVO.setFileName(originalName);
        return fileVO;
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 上传组件逻辑封装

```javascript
/**
 * 文件上传逻辑 (纯 JS)
 * 强制要求：禁止使用 TypeScript
 */
import { ref } from 'vue';
import { fileApi } from '@/api/file';

export function useFileUpload() {
  const uploading = ref(false);
  const progress = ref(0);

  const executeUpload = async (file) => {
    uploading.value = true;
    try {
      const formData = new FormData();
      formData.append('file', file);
      const response = await fileApi.upload(formData, (p) => {
        progress.value = p;
      });
      return response.data;
    } finally {
      uploading.value = false;
    }
  };

  return {
    uploading,
    progress,
    executeUpload
  };
}
```

## 四、UI 表现与对齐规范 (DingTalk Style)

### 4.1 文件消息气泡 (File Message Bubble)
- **卡片样式**：
  - **最大宽度**：`280px`。背景色 `#FFFFFF`，边框 `1px solid rgba(23, 26, 29, 0.08)`。
  - **布局**：左右结构。左侧为文字信息（文件名、大小），右侧为文件类型图标。
  - **内边距**：`12px`。
- **图标索引**：
  - `PDF`：红色 PDF 图标。
  - `Word`：蓝色 Word 图标。
  - `Excel`：绿色 Excel 图标。
  - 其他：通用灰色文档图标。
- **进度条**：
  - **位置**：气泡底部贴边。
  - **样式**：高度 `2px`，色值 `#0089FF`。仅在上传/下载中显示。

### 4.2 侧边栏文件列表 (File Drawer)
- **列表项**：
  - 高度 `52px`，包含文件名、大小、发送者、发送时间。
  - 悬停（Hover）显示下载/预览按钮。
- **批量操作**：支持多选文件进行批量下载，复选框色值为 `#0089FF`。

### 4.3 交互状态反馈
- **上传中**：图标覆盖半透明遮罩，居中显示百分比。
- **发送失败**：气泡左侧出现红色感叹号按钮，点击触发重发。
