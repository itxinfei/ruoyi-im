# 后台执行任务卡（第一批）

> 来源基线：`docs/后台管理系统功能与页面设计需求.md`  
> 目标：将后台管理系统从“基础可用”推进到“可发布验收”。

---

## 任务清单（按执行顺序）

1. `IM-ADM-001` 系统配置页面与参数治理闭环（P0）
2. `IM-ADM-002` 批量操作结果标准化（P0）
3. `IM-ADM-003` 用户管理增强（P1）
4. `IM-ADM-004` 后台审计日志可视化（P1）

---

## IM-ADM-001 系统配置页面与参数治理闭环（P0）

- 目标：新增“系统配置”页面，打通参数查询/编辑/保存/回滚。
- 前端范围：
  - `ruoyi-im-web/src/router/index.js`
  - `ruoyi-im-web/src/views/admin/AdminLayout.vue`
  - `ruoyi-im-web/src/views/admin`（新增 `SystemConfig.vue`）
  - `ruoyi-im-web/src/api/admin.js`
- 后端范围：
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/controller/admin/ImSystemConfigAdminController.java`
  - 相关 service/mapper（按现有实现补齐）
- 验收：
  - 菜单可达，页面可读写配置
  - `ADMIN` 与 `SUPER_ADMIN` 写权限边界正确
  - 参数变更有明确成功/失败反馈

## IM-ADM-002 批量操作结果标准化（P0）

- 目标：统一批量接口返回结构与前端反馈展示。
- 前端范围：
  - `ruoyi-im-web/src/views/admin/UserManagement.vue`
  - `ruoyi-im-web/src/views/admin/GroupManagement.vue`
  - `ruoyi-im-web/src/views/admin/MessageManagement.vue`
- 后端范围：
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/controller/admin/ImGroupAdminController.java`
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/controller/admin/ImMessageAdminController.java`
- 验收：
  - 批量接口统一返回 `successCount/failedCount/failedItems`
  - 前端统一显示“成功数+失败数”
  - 失败项原因可读

## IM-ADM-003 用户管理增强（P1）

- 目标：补角色变更入口与用户详情抽屉。
- 前端范围：
  - `ruoyi-im-web/src/views/admin/UserManagement.vue`
  - `ruoyi-im-web/src/api/admin.js`
- 后端范围：
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/controller/admin/ImUserAdminController.java`
- 验收：
  - `SUPER_ADMIN` 可修改角色，`ADMIN` 不可见/不可写
  - 用户详情抽屉可查看基础资料与最近活动
  - 危险操作二次确认

## IM-ADM-004 后台审计日志可视化（P1）

- 目标：提供后台操作审计页面（基础版）。
- 前端范围：
  - `ruoyi-im-web/src/views/admin`（新增 `AuditLog.vue` 或等价页面）
  - `ruoyi-im-web/src/api/admin.js`
- 后端范围：
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/controller/admin`（新增或补齐审计接口）
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/service`（审计查询服务）
- 验收：
  - 可按操作人/模块/时间查询
  - 可查看对象、动作、结果、时间
  - 查询分页稳定，空态正确

---

## 执行规则

1. 每次只推进一个 `task_id`，完成后再切下一个。
2. 每个任务必须回写 `docs/llm-tasks.yaml` 的状态。
3. 每个任务必须附带验证命令结果：
   - 前端：`npm run build`
   - 后端：`mvn -DskipTests compile`
4. 禁止保留“开发中”入口作为交付结果。
