## 二、后端实现 (JDK 1.8 规范)

### 2.1 全文搜索逻辑 (MySQL 5.7)

```java
@Service
@RequiredArgsConstructor
public class ImSearchServiceImpl implements ImSearchService {

    private final ImMessageMapper messageMapper;

    /**
     * 搜索消息内容 (高亮处理)
     * 遵循 JDK 1.8，严禁 var
     */
    public List<MessageVO> searchMessages(String keyword, Long conversationId) {
        LambdaQueryWrapper<ImMessage> wrapper = Wrappers.lambdaQuery();
        wrapper.eq(ImMessage::getConversationId, conversationId);
        // 使用 MATCH AGAINST 实现全文检索
        wrapper.apply("MATCH(content) AGAINST({0} IN BOOLEAN MODE)", keyword);
        
        List<ImMessage> list = messageMapper.selectList(wrapper);
        return convertToVO(list, keyword);
    }
}
```

### 3.1 搜索防抖与跳转逻辑

```javascript
/**
 * 搜索逻辑封装 (纯 JS)
 */
import { ref } from 'vue';
import { searchApi } from '@/api/search';

export function useSearch() {
  const keyword = ref('');
  const results = ref([]);
  let timer = null;

  const processSearchInput = () => {
    if (timer) clearTimeout(timer);
    timer = setTimeout(async () => {
      if (!keyword.value) return;
      const response = await searchApi.global(keyword.value);
      results.value = response.data || [];
    }, 300);
  };

  const goToResult = (item) => {
    // 根据 item.type 进行路由跳转逻辑
  };

  return {
    keyword,
    results,
    processSearchInput,
    goToResult
  };
}
```

## 四、UI 表现与对齐规范 (DingTalk Style)

### 4.1 搜索输入框 (L2 Session Bar)
- **静态样式**：高度 `32px`，背景色 `rgba(23, 26, 29, 0.04)`，圆角 `4px`。
- **Focus 态**：背景色变为 `#FFFFFF`，边框变为 `1px solid #0089FF`，产生极轻微的蓝色投影。
- **清除按钮**：输入内容后，右侧出现圆形的 `×` 图标，点击清空。

### 4.2 搜索结果展示 (Search Results)
- **分类策略**：结果按“联系人”、“群组”、“聊天记录”进行分组展示。
- **关键词高亮**：匹配到的文字必须使用钉钉标准高亮色 `#FF9000` (橙色)，且不带背景。
- **信息密度**：
  - 列表项高度 `52px`。
  - 标题 `14px`，摘要 `12px` (单行溢出省略)。
- **空状态**：搜索无结果时，展示居中的插画（灰色）及提示语“未搜索到相关内容”。

### 4.3 全局搜索弹窗
- **触发**：由左侧 L1 导航栏的搜索图标或 `Ctrl + F` 触发。
- **样式**：居中大弹窗 (`800x600`)，支持分类 Tab 切换，背景使用 `var(--dt-bg-body)` (#F2F4F5)。
