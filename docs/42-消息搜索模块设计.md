## 二、后端实现 (JDK 1.8 规范)

### 2.1 全文搜索逻辑 (MySQL 5.7)

```java
@Service
@RequiredArgsConstructor
public class ImSearchServiceImpl implements ImSearchService {

    private final ImMessageMapper messageMapper;

    /**
     * 搜索消息内容 (高亮处理)
     * 遵循 JDK 1.8，严禁 var
     */
    public List<MessageVO> searchMessages(String keyword, Long conversationId) {
        LambdaQueryWrapper<ImMessage> wrapper = Wrappers.lambdaQuery();
        wrapper.eq(ImMessage::getConversationId, conversationId);
        // 使用 MATCH AGAINST 实现全文检索
        wrapper.apply("MATCH(content) AGAINST({0} IN BOOLEAN MODE)", keyword);
        
        List<ImMessage> list = messageMapper.selectList(wrapper);
        return convertToVO(list, keyword);
    }
}
```

export function useSearch() {
  const keyword = ref('');
  const results = ref([]);
  let timer = null;

  const processSearchInput = () => {
    if (timer) clearTimeout(timer);
    timer = setTimeout(async () => {
      if (!keyword.value) return;
      const response = await searchApi.global(keyword.value);
      results.value = response.data || [];
    }, 300);
  };

  const goToResult = (item) => {
    // 根据 item.type 进行路由跳转逻辑
  };

    keyword,

    processSearchInput,

}
