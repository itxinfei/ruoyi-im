## IM-P0-001 消息二级菜单筛选闭环

- 目标：完成“全部、未读、置顶、免打扰、群聊、文件链接”六类筛选。
- 前端范围：
  - `ruoyi-im-web/src/views/SessionPanel.vue`
  - `ruoyi-im-web/src/store/modules/im-session.js`
  - `ruoyi-im-web/src/api/im/conversation.js`
- 后端范围：
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/controller/ImConversationController.java`
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/service/impl/ImConversationServiceImpl.java`
- 验收：
  - 菜单切换即生效，结果准确
  - 刷新后状态可恢复
  - 错误有提示，不出现无响应

## IM-P0-002 会话详情弹窗字段对齐

- 目标：统一会话详情字段，消除 targetId/peerName/avatar 错位。
- 范围：

  - `ruoyi-im-web/src/views/MainPage.vue`
  - `ruoyi-im-web/src/api/im/conversation.js`
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/vo/conversation/ImConversationVO.java`
- 验收：
  - 详情页信息正确
  - 详情页发消息/发起通话可用
  - 无空字段展示

## IM-P0-003 文件预览弹窗闭环

- 范围：

## IM-P0-004 通话弹窗状态机定版

  - 异常断线可收敛状态
  - 无重复弹窗

## IM-P0-005 `sql/im.sql` 与 Mapper 一致性核查

- 目标：以 `sql/im.sql` 为单一事实源，完成表结构与 Mapper 对齐。
- 范围：
  - `sql/im.sql`
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/domain`
  - `ruoyi-im-api/src/main/java/com/ruoyi/im/mapper`
  - `ruoyi-im-api/src/main/resources/mapper`
- 验收：
  - 输出差异清单（字段、类型、默认值、约束、索引）
  - 聊天核心链路相关 Mapper 修复后可编译
  - 明确高风险变更回滚方案

---

## 执行规则

1. 每次只做一个 task_id。
2. 每次提交必须包含验证结果。
3. Java 代码必须符合 JDK8 规范，禁止魔法值与复杂表达式链。
4. UI 改动必须联动业务逻辑，禁止只改样式。
