## 1. 消息气泡全态 UI 与交互 (Message Bubble) - 【最关键部分】

### 1.0 气泡组件结构 (必须严格按照此结构)

```vue
<!-- 消息气泡组件 - ChatMessageBubble.vue -->
<template>
  <div
    class="message-bubble-wrapper"
    :class="{ 'is-own': isOwn, 'is-group': isGroup }"
    @mouseenter="onMouseEnter"
    @mouseleave="onMouseLeave"
  >
    <!-- 头像：仅非连续消息显示 -->
    <el-avatar
      v-if="showAvatar"
      :size="36"
      :src="message.senderAvatar"
      class="bubble-avatar"
    />

    <!-- 消息内容区 -->
    <div class="bubble-content">
      <!-- 发送者名称：仅群聊且非连续消息显示 -->
      <div v-if="showSenderName" class="bubble-sender">
        {{ message.senderName }}
      </div>

      <!-- 气泡主体 -->
      <div class="bubble-body" :class="{ 'is-own': isOwn }">
        <div class="bubble-text">{{ message.content }}</div>

        <!-- 发送状态：仅发送方显示 -->
        <span v-if="isOwn" class="bubble-status">
          {{ statusText }}
        </span>
      </div>

      <!-- 已读/未读状态 -->
      <div v-if="isOwn && showReadStatus" class="bubble-read-status">
        {{ readStatusText }}
      </div>
    </div>

    <!-- 操作工具条：hover 时显示 -->
    <div v-show="isHovered" class="bubble-action-bar">
      <i class="el-icon-copy-document" @click="onCopy" />
      <i class="el-icon-share" @click="onForward" />
      <i class="el-icon-delete" @click="onDelete" />
    </div>
  </div>
</template>
```

### 1.2 气泡圆角逻辑 - 【必须严格遵守】

```
钉钉气泡圆角不是四个角都一样！

发送方 (Me):  ┌────┐          接收方 (Other):  ┌────┐
              │    │                         │    │
              └────┘                         └────┘
               ↑                              ↑
          右下8px 左下8px                  左上8px 左下8px
          右上4px 左上8px                  右上8px 右上8px
```

| 方向 | 发送方 (is-own) | 接收方 (!is-own) |
|------|-----------------|------------------|
| 右上角 | **4px** (尖) | 8px |
| 右下角 | 8px | 8px |
| 左下角 | 8px | 8px |
| 左上角 | 8px | **4px** (尖) |

```scss
// 发送方气泡圆角
.bubble-body.is-own {
  border-radius: 8px 4px 8px 8px;  // clockwise: 右上4px, 右下8px, 左下8px, 左上8px
}

// 接收方气泡圆角
.bubble-body:not(.is-own) {
  border-radius: 4px 8px 8px 8px;  // clockwise: 右上8px, 右下8px, 左下8px, 左上4px
}
```

### 1.3 头像与气泡对齐规则 - 【核心关键】

```
对齐方式：头像的【顶部】必须与气泡的【顶部】水平对齐

错误做法：
  ❌ 头像垂直居中于消息行
  ❌ 头像底部与气泡底部对齐

正确做法：
  ✅ 头像顶部与气泡第一行文字顶部对齐
```


### 1.4 连续消息压缩规则

```vue
<!-- 连续消息判断逻辑 -->
<script>
computed: {
  // 连续消息：同一发送者、同一分钟内、且不是最后一条
  isConsecutive() {
    const prev = this.getPrevMessage()
    if (!prev) return false
    if (prev.senderId !== this.message.senderId) return false
    const timeDiff = this.message.timestamp - prev.timestamp
    return timeDiff < 60000  // 1分钟内
  },

  showAvatar() {
    return !this.isConsecutive  // 连续消息隐藏头像
  },

  showSenderName() {
    return !this.isConsecutive && this.isGroup  // 群聊且非连续才显示昵称
  }
}
</script>
```

```scss
// 连续消息样式
.message-bubble-wrapper {
  &.is-consecutive {
    padding-top: 2px;       // 连续消息间距压缩到 2px
    gap: 8px;              // 头像间距压缩

    .bubble-avatar {
      visibility: hidden;   // 隐藏但仍占位，保持对齐
      height: 0;
    }
  }
}
```

### 1.5 发送状态闭环 - 【必须实现完整状态机】

```typescript
// 消息状态枚举
enum MessageStatus {
  PENDING = 'PENDING',      // 待发送
  SENDING = 'SENDING',      // 发送中
  SENT = 'SENT',            // 已发送
  DELIVERED = 'DELIVERED',  // 已送达
  FAILED = 'FAILED'         // 发送失败
}

// 状态流转
// PENDING → SENDING → SENT → DELIVERED
//                    ↘ FAILED
```

```vue
<!-- 状态显示模板 -->
<span class="bubble-status" :class="statusClass">
  <i v-if="status === 'SENDING'" class="el-icon-loading" />
  <i v-else-if="status === 'FAILED'" class="el-icon-warning" />
  <span v-else>{{ statusText }}</span>
</span>
```


## 5. UI 验收负面清单 (Negative Constraints)

### 5.1 禁止行为汇总

| 禁止项 | 错误示例 | 正确做法 |
|--------|----------|----------|
| 圆形头像 | `border-radius: 50%` | `border-radius: 4px` |
| 大圆角 | `border-radius: 16px` | `border-radius: 4px` 或 `8px` |
| 气泡渐变 | `background: linear-gradient(...)` | `background: #277EFB` 或 `#FFF` |
| 发送图标大阴影 | `box-shadow: 0 4px 12px` | `box-shadow: none` 或极轻阴影 |
| 全局滚动条 | `body { overflow: auto }` | `overflow: hidden` |
| 状态复杂动画 | `transition: 500ms` | `transition: 200ms` |
| 浮动按钮 | `position: absolute` 主布局 | `flex` 布局 |

### 5.2 颜色白名单
