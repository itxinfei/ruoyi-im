# 18 平台模块与管理后台需求

> **用途**：定义除 IM 核心外的各功能模块及管理员后台的需求逻辑与 UI 设计规范。
> **更新时间**：2026-03-21

---

## 一、平台协同模块 (Client Modules)

> 这些模块面向普通用户，是管理后台之外的业务功能模块。
