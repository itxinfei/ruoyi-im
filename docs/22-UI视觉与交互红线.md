# 22 UI 视觉与交互红线 (Visual Checklist)

> **定位**：定义 IM 核心页面的“钉钉感”验收标准，任何 UI 优化必须通过此红线。

---

## 1. 会话列表 (Session List) 红线

- **[L1] 高度固定**：每个会话项高度必须固定为 `64px`，严禁随内容撑开。
- **[L2] 标题截断**：长群名或长昵称必须使用单行省略号（`text-overflow: ellipsis`）。
- **[L3] 未读数样式**：数字未读数必须是带微圆角的红色背景（钉钉风格），严禁出现正圆大红点。
- **[L4] 选中态**：当前选中会话背景色为 `#E6F7FF` 或浅灰色，左侧带一条 `2px` 宽的蓝色指示条。

## 2. 消息展示 (Message Bubble) 红线

- **[M1] 气泡对齐**：
    - 发送方：头像在右，气泡顶部与头像顶部在**同一水平线**。
    - 接收方：头像在左，气泡顶部与头像顶部在**同一水平线**。
- **[M2] 圆角规范**：
    - 接收方：圆角 `4px 12px 12px 12px`（左上角为尖角）。
    - 发送方：圆角 `12px 4px 12px 12px`（右上角为尖角）。
- **[M3] 消息间距**：
    - 同一分钟内、同一发送者的连续消息，间距为 `4px` 且不重复显示头像和昵称。
    - 异侧消息或不同批次消息，间距为 `16px`。
- **[M4] 交互反馈**：
    - 消息操作栏（复制、转发、撤回）必须在 `hover` 时才浮现，严禁常驻。

## 3. 输入区 (Input Area) 红线

- **[I1] 边框一致性**：输入框获取焦点时，整体边框（包含工具栏）变为蓝色。
- **[I2] 工具栏间距**：图标按钮间距固定为 `12px`，图标尺寸统一为 `20x20`。
- **[I3] 发送按钮**：仅当输入框有文字或文件待发送时，发送按钮才高亮；否则置灰。

## 4. 颜色与字体红线

- **[C1] 背景分层**：
    - 会话栏背景：`#F7F7F7`
    - 聊天区背景：`#FFFFFF`
    - 详情栏背景：`#FFFFFF` (左侧带 `1px` 深灰色分割线)
- **[C2] 字体规范**：
    - 主文字：`14px`, `#1F1F1F`
    - 次要文字（最后消息、时间）：`12px`, `#8C8C8C`
