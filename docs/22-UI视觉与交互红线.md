# 22 UI 视觉与交互红线 (UI Quality Checklist)

> **定位**：定义 IM 核心页面的"对齐钉钉"验收硬指标。任何代码生成的 UI 必须通过此清单检查，否则视为不合格。
> **版本**：v2.0 - 可执行验收清单

---

## 1. 核心骨架 (Skeleton) 红线

### [S1] 三栏布局固定化
- [ ] L1 侧边栏宽度固定 `64px`
- [ ] L2 会话列表宽度固定 `250px`
- [ ] L3 聊天区域 `flex: 1`，最小宽度 `480px`
- [ ] **禁止**使用百分比布局导致宽度无限拉伸

```scss
// ✅ 正确
.im-session-list { width: 250px; flex-shrink: 0; }

// ❌ 错误
.im-session-list { width: 25%; flex-shrink: 0; }  // 禁止百分比
```

### [S2] 根滚动条禁令
- [ ] 全局 `body` 和 `#app` 设置 `overflow: hidden`
- [ ] 仅允许 L2 列表、L3 聊天区域局部滚动
- [ ] 滚动条样式：宽度 `6px`，滑块 `rgba(0,0,0,0.2)`

### [S3] 分割线规范
- [ ] 三栏之间的分割线颜色必须是 `rgba(23, 26, 29, 0.08)`
- [ ] 分割线宽度 `1px`

## 2. 消息气泡 (Message Bubble) 精准红线

### [M1] 气泡圆角逻辑 - 【最高优先级】

**发送方 (is-own):**
```scss
border-radius: 8px 4px 8px 8px;
// 右上角 4px (尖角)，其余 8px
```

**接收方 (!is-own):**
```scss
border-radius: 4px 8px 8px 8px;
// 左上角 4px (尖角)，其余 8px
```

**验收要点：**
- [ ] 四个角半径**不是全部相同**
- [ ] 发送方右上角**必须是 4px**（尖角感）
- [ ] 接收方左上角**必须是 4px**（尖角感）

### [M2] 头像与气泡对齐 - 【核心关键】

- [ ] 头像**顶部**与气泡**第一行文字顶部**水平对齐
- [ ] **禁止**使用 `align-items: center` 导致头像垂直居中
- [ ] 头像必须是**圆角矩形**（`border-radius: 4px`），**不是圆形**

```scss
// ✅ 正确：顶部对齐
.message-bubble-wrapper {
  display: flex;
  align-items: flex-start;  // 不是 center
}

// ❌ 错误
.message-bubble-wrapper {
  display: flex;
  align-items: center;  // 禁止垂直居中
}
```

### [M3] 连续消息压缩

- [ ] 同一发送者、1分钟内的连续消息，**隐藏头像和昵称**
- [ ] 连续消息间距压缩至 `4px`

```javascript
// 判断逻辑
const isConsecutive = prev &&
//   prev.senderId === current.senderId &&
//   current.timestamp - prev.timestamp < 60000;
```

### [M4] 气泡内边距

- [ ] 气泡 padding：**上下 8px，左右 12px**
- [ ] 气泡最大宽度：`70%`
- [ ] 气泡字号：`14px`
- [ ] 气泡行高：`1.5`

### [M5] 气泡颜色

- [ ] 发送方背景：`#277EFB`，文字白色
- [ ] 接收方背景：`#FFFFFF`，文字 `#171A1D`，边框 `1px solid rgba(23, 26, 29, 0.08)`

### [M6] 操作工具条

- [ ] **hover 时浮现**，严禁常驻显示
- [ ] 工具条位置：发送方在气泡**左侧**，接收方在气泡**右侧**
- [ ] 工具条圆角：`4px`
- [ ] 工具条阴影：`0 2px 8px rgba(0,0,0,0.08)`
- [ ] 工具条图标尺寸：`20x20px`，hover 时变蓝

## 3. 颜色与字体 (Colors & Fonts) 红线

### [C1] 品牌蓝锁定

- [ ] 主色调统一使用 `#277EFB`（钉钉官方蓝）
- [ ] **禁止**使用 `#0089FF`（旧主色）
- [ ] **禁止**使用 `#1677FF`、`#1890ff`（非标蓝色）

### [C2] 文本灰度分层

| 用途 | 色值 | CSS 变量 |
|------|------|----------|
| 核心正文 | `#171A1D` | `--dt-text-primary` |
| 辅助/摘要 | `rgba(23, 26, 29, 0.4)` | `--dt-text-tertiary` |

- [ ] 无头像时使用钉钉三色（蓝、橙、绿）随机底色
- [ ] 头像圆角：`4px`（**不是圆形**）

## 7. 禁止行为 (Hard Bans) 汇总

| 禁止项 | 违规示例 | 正确做法 |
|--------|----------|----------|
| 使用 `el-row/col` 主布局 | `<el-row>` | `display: flex` |
| 超过 12px 圆角 | `border-radius: 16px` | `border-radius: 4px` 或 `8px` |
| 消息区横向滚动 | `overflow-x: auto` | `overflow-x: hidden` |
| 全局滚动条 | `body { overflow }` | `overflow: hidden` |
| 复杂阴影 | `box-shadow: 0 4px 12px` | `box-shadow: none` 或极轻阴影 |
| 渐变背景 | `background: linear-gradient` | 纯色背景 |
| 悬浮位移/缩放 | `transform: scale(1.05)` | 禁止 |
| 圆形头像 | `border-radius: 50%` | `border-radius: 4px` |

## 8. 验收自检流程

### 代码生成后，必须逐项检查：

1. **气泡圆角**：是否四个角不同？发送方右上/接收方左上是否为 4px？
2. **头像对齐**：是否为 `align-items: flex-start`？头像是否是 4px 圆角矩形？
3. **工具栏状态**：操作条是否 hover 时浮现？是否在正确侧边显示？
4. **颜色规范**：是否使用 `#277EFB` 而非其他蓝色？
5. **间距系统**：所有间距是否为 4/8/12/16/24/32px？
6. **滚动条**：全局是否禁止滚动？局部滚动条是否为 6px 宽？

### 验收通过标准

- [ ] 上述 7 大类 30+ 检查项全部通过
- [ ] 无任何禁止行为
- [ ] 所有硬编码色值已替换为 CSS 变量
