# 28 通讯录模块UI设计规范

> **用途**：定义通讯录模块的页面布局、组件交互及视觉标准。  
> **设计基准**：钉钉 Windows 8.2.0 桌面端，遵循 `docs/27-PC版钉钉UI设计蓝图.md`。

---

## 1. 页面定位与模板

*   **模块 ID**：`contacts`
*   **页面模板**：列表-详情协同页 (`L1 + L2 + L3 + Drawer`)
*   **主要任务**：
    1.  浏览组织架构树。
    2.  快速搜索联系人、部门或群组。
    3.  查看成员详情并一键发起聊天。
    4.  管理好友关系（添加/删除）。

---

## 2. 空间布局 (Layout)

### 2.1 整体骨架
*   **L1 (一级导航)**：深色侧边栏，宽度 `64px`。
*   **L2 (二级导航/树)**：宽度 `250px`，固定布局，承载 Tab 切换与目录树。
*   **L3 (主视图区)**：`flex: 1`，显示当前选中节点（部门/群组）的成员列表。
*   **Drawer (详情抽屉)**：从右侧弹出，宽度 `360px`，承载成员/群组详细资料。

### 2.2 搜索与 Tab 区 (L2 Header)
*   **搜索框**：高度 `32px`，居中对齐，背景色 `rgba(23, 26, 29, 0.04)`。
*   **Tab 切换**：
    *   项：组织架构、我的群组、我的好友、外部联系人。
    *   样式：文字高度 `40px`，激活项带 `2px` 品牌蓝下划线。

---

## 3. 组件级设计规范

### 3.1 组织架构树 (Dept Tree)
*   **行高**：`36px`。
*   **缩进**：每层级递进 `16px`。
*   **图标**：
    *   部门：`Folder` 图标，默认色 `--dt-text-icon`。
    *   展开状态：三角形箭头，顺时针旋转 `90deg`。
*   **交互**：Hover 背景色 `var(--dt-bg-hover)`，选中背景色 `var(--dt-bg-session-active)`，文字加粗。

### 3.2 成员列表 (Member List)
*   **表头**：高度 `40px`，背景色 `var(--dt-bg-body)`，含“姓名”、“职位”、“部门”、“手机号”。
*   **列表行**：
    *   高度：`56px`。
    *   头像：`36px * 36px`，圆角 `4px` (var(--dt-radius-sm))。
    *   文本：姓名（Primary, 14px），职位（Secondary, 12px）。
*   **操作项**：右侧显示“发消息”按钮（默认隐藏，Hover 出现）。

### 3.3 联系人详情卡片 (User Profile Drawer)
*   **头部**：大幅背景（品牌浅蓝），头像 `64px` 居中，姓名加粗 `18px`。
*   **信息块**：
    *   部门、职位、手机、邮箱、工号。
    *   标签：`Profile-Item` 12px Tertiary。
    *   值：`Profile-Value` 14px Primary，支持一键复制。
*   **底部操作栏**：
    *   `[发消息]`：主按钮，品牌蓝。
    *   `[视频通话] / [音频通话]`：次按钮，线性边框。

---

## 4. 状态清单 (States)

| 状态 | 表现 |
|-----|------|
| **Loading** | 树结构加载时使用骨架屏，列表加载使用顶部进度条。 |
| **Empty** | 搜索无结果时显示“未找到匹配的联系人”，带“清除搜索”按钮。 |
| **Selected** | 树节点背景色 `#E6F2FF`，右侧列表高亮显示所属部门路径。 |
| **Hover** | 列表行出现“发消息”和“收藏”快捷图标。 |

---

## 5. 交互路径 (Interactions)

1.  **发起聊天**：点击列表行 -> 右侧弹出抽屉 -> 点击“发消息” -> 路由跳转至 `/chat?id=xxx` 并选中该会话。
2.  **树联动**：点击左侧“技术部” -> 右侧列表刷新为技术部全员 -> 顶部面包屑显示“总公司 / 技术部”。
3.  **全局搜索**：输入“张三” -> L2 区域切换为“搜索结果”视图 -> 分类显示“联系人(3)”、“部门(0)”、“群组(1)”。

---

## 6. 验收点 (AC)

- [ ] AC1: 部门树支持无限级嵌套，且点击展开/收起动效不卡顿。
- [ ] AC2: 成员列表支持按姓名 A-Z 排序，支持首字母导航（如有）。
- [ ] AC3: 详情抽屉滑出平滑，且在点击遮罩或外部时自动收起。
- [ ] AC4: 所有颜色和间距 100% 引用 `design-tokens.scss` 变量。
