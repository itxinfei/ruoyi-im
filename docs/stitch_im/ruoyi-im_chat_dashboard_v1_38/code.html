关于2023年国庆节放假安排的通知

行
人事行政部 · 王大伟 • 3小时前发布

各位同事：

根据国家法定节假日放假安排，结合公司实际生产经营情况，现将2023年国庆节放假及调休安排通知如下：

一、放假时间

9月29日（星期五）至10月6日（星期五）放假调休，共8天。

二、调休上班

10月7日（星期六）、10月8日（星期日）正常上班。

请各部门提前做好工作安排，放假前注意关闭电源、锁好门窗，做好安全防疫工作。

若依IM 人事行政部

2023年9月25日
