# 54 验收用例清单

> 用途：把 IM 与平台模块的人工验收步骤写清，方便自己和大模型按 checklist 复核。
> 说明：本文件只记录可重复验证的验收用例，未完全实现的不写。

## 1. 使用说明

1. 每次完成一个任务后，对照本清单核对对应项。
2. 本清单按模块分类，粒度控制在“一个页面/一个能力”。
3. 若验收涉及前端与后端，需同时验证 UI 行为和接口数据。
4. 对于 `unknown` 项目，先确认后再打勾，不允许无凭据核实。

## 2. IM 核心路径

### 2.1 会话列表

1. 进入消息页后默认展示会话列表。
2. 点击会话项后聊天区切换且当前选中态高亮。
3. 会话列表显示未读数，空状态展示提示。
4. 会话搜索可以过滤结果。

### 2.2 聊天区与气泡

1. 进入会话后滚到最后一条消息。
2. 发送方气泡在前、头像在后、最大宽度 70%。
3. 同侧间距 4px，异侧 16px。
4. 时间分割线不造成双重留白。
5. 引用/回复/转发菜单可进入实际流程。

### 2.3 输入区

1. Enter 发送，Ctrl+Enter 换行。
2. 图片/文件按钮可选本地文件并上传。
3. 粘贴图片/文件会触发上传。
4. 发送失败时提示并支持重试。
5. 群聊 `@` 可以选人。

### 2.4 搜索

1. 会话栏搜索可以过滤当前会话列表。
2. 全局搜索页出现消息、联系人、群组、文件结果。
3. 会话内搜索可以定位到消息。

### 2.5 群详情与群文件

1. 打开群详情可以看到名称、成员、公告。
2. 详情内有群文件入口，打开面板能加载文件列表。
3. 文件可按类别筛选，支持上传与下载。

### 2.6 通话

1. 聊天头部语音/视频按钮可发起通话。
2. 对端可接听、拒绝、挂断，异常有提示。
3. 通话状态流（calling/connecting/talking/ended）可切换。

### 2.7 收藏与链接卡片

1. 消息右键或 hover 菜单能收藏/取消收藏。
2. 收藏列表可从消息页入口打开，显示已收藏消息。
3. 发送 URL 后展示链接卡片，解析失败则降级普通链接。

## 3. 平台模块

### 3.1 通讯录

1. 左侧导航“通讯录”可进入。
2. 可通过树状部门结构浏览成员。
3. 点击成员可跳转个人详情/发起聊天。

### 3.2 搜索模块

1. 左侧导航“搜索”或全局快捷键可进入。
2. 输入关键词后数据分类展示。
3. 结果可跳转到对应模块。

### 3.3 工作台

1. “工作台”页面可进入。
2. 显示真实可点击的子模块入口。
3. 点击任意入口成功跳转对应页。

### 3.4 云盘与文档

1. 云盘列表能加载文件。
2. 可上传/下载文件并显示进度。
3. 搜索可定位结果。

### 3.5 日历、待办、审批

1. 各模块能够打开对应页面。
2. 日历支持查看、新建、编辑、删除日程。
3. 待办可以新建与完成任务，能搜索或筛选。
4. 审批支持发起、查看状态与查看详情。

### 3.6 邮箱、AI 助手、设置、个人资料、管理入口

1. 各入口可进入并展示基础数据。
2. 邮箱可查看邮件列表与详情。
3. AI 助手能发起问答并反馈状态。
4. 设置/个人资料可修改并保存至少一类字段。
5. 管理入口仅向管理员显示，包含用户与系统配置页。
