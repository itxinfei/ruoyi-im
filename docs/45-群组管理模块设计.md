## 二、后端实现 (JDK 1.8 规范)

### 2.1 核心成员校验逻辑

```java
@Service
@RequiredArgsConstructor
public class ImGroupServiceImpl implements ImGroupService {

    private final ImGroupMapper groupMapper;
    private final ImGroupMemberMapper memberMapper;

    /**
     * 校验用户是否有管理员权限
     * 遵循 JDK 1.8，严禁 var
     */
    public boolean isAdmin(Long groupId, Long userId) {
        ImGroupMember member = memberMapper.selectByGroupIdAndUserId(groupId, userId);
        if (Objects.isNull(member)) {
            return false;
        }
        String role = member.getRole();
        return "OWNER".equals(role) || "ADMIN".equals(role);
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 群详情抽屉逻辑封装

```javascript
/**
 * 群详情逻辑封装 (纯 JS)
 * 强制要求：禁止使用 TypeScript
 */
import { ref, computed } from 'vue';
import { groupApi } from '@/api/group';

export function useGroupDetail(groupId) {
  const group = ref({});
  const members = ref([]);

  const loadGroup = async () => {
    const response = await groupApi.getDetail(groupId);
    group.value = response.data || {};
  };

  const loadMembers = async () => {
    const response = await groupApi.getMembers(groupId);
    members.value = response.list || [];
  };

  const isOwner = computed(() => {
    const currentUserId = 1; // 示例 ID
    return group.value.ownerId === currentUserId;
  });

  return {
    group,
    members,
    isOwner,
    loadGroup,
    loadMembers
  };
}
```
