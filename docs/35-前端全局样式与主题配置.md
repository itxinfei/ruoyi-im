# 35 前端全局样式与主题配置 (Theme & Styles)

> **用途**：统一内网环境下 UI 的视觉基因，确保所有样式基于项目实际的 CSS 变量和设计令牌（design‑tokens），防止硬编码和错误的 Tailwind 引用。

---

## 1. 样式体系概述

项目采用 **CSS 变量 + SCSS 设计令牌** 的方式管理颜色、间距、圆角等全局视觉属性。所有组件样式应通过 `var(--xxx)` 或 SCSS 变量引用，避免直接写死颜色或尺寸。

- **设计令牌文件**：`src/styles/design-tokens.scss` 定义了颜色、字体、阴影等变量，并在 `:root` 中导出为 CSS 变量。
- **全局样式入口**：`src/styles/global.scss` 负责基础重置、Element Plus 覆盖与 `.u-*` 辅助类。
- **Tailwind 扩展层**：`src/styles/tailwind-overrides.scss` 通过 `@layer utilities/components` 提供局部工具类与组件样式扩展。
- **Tailwind 使用边界**：项目仅保留 Tailwind 的工具类与 `@apply` 扩展能力，不作为独立视觉体系；文档中不应出现 `bg-primary-sidebar` 这类脱离 token 体系的硬编码示例。

## 3. 组件级样式指南 (Implementation Logic)

为了让 UI 不产生“廉价感”，AI 在编写 Vue 组件时必须遵循以下逻辑：

### 3.1 会话列表项 (Session Item)
- **高度**：固定 `56px`（钉钉 8.2 标准，**不是 64px**，权威依据：`00-钉钉8.2像素级权威基线.md` §1.2）。
- **背景**：默认透明，Hover 使用 `var(--dt-bg-session-hover)`，选中时使用 `var(--dt-bg-session-active)` 且左侧带一条 `3px` 蓝边。
- **摘要文字**：必须限制为 `ellipsis` (单行溢出省略)，使用 `var(--dt-text-tertiary)`。

### 3.2 消息气泡 (Message Bubble)
- **间距**：同侧气泡间距 `4px`，异侧 `16px`。
- **头像**：圆角固定 `4px`（不要用圆形，圆形是移动端基因），尺寸 `36x36`。
- **文字**：行高 `1.5`，字体大小 `14px`。

### 3.3 全局边框
- 所有卡片和面板的边框必须是 `var(--dt-line-width) solid var(--dt-border-color)`。
- 禁止使用 `box-shadow` 做大面积投影，仅允许在悬浮菜单使用极轻微阴影。

## 4. 全局实用类 (Utility Classes)

为配合 CSS 变量，项目当前提供两类可复用样式层：

- `src/styles/global.scss` 中的 `.u-*` 辅助类，用于基础布局和文本截断。
- `src/styles/tailwind-overrides.scss` 中的 `@layer utilities/components`，用于滚动条、卡片、按钮等有限扩展。

```scss
/* src/styles/global.scss */
.u-flex {
  display: flex;
}

.u-flex-center {
  align-items: center;
  justify-content: center;
}

.u-text-ellipsis {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

/* src/styles/tailwind-overrides.scss */
@layer utilities {
  .scrollbar-thin {
    scrollbar-width: thin;
  }

  .card {
    @apply bg-white dark:bg-slate-800 rounded-lg border border-slate-200 dark:border-slate-700;
  }
}
```

> **变量命名规范**：
> - `--dt-*`：设计令牌（Design Token），定义基础视觉基因（颜色、间距、圆角）
> - `--im-*`：组件变量，在 `--dt-*` 基础上定制特定组件外观
> - **统一要求**：文档示例统一使用 `--dt-*` 前缀，组件级变量使用 `--im-*`
> - **注意**：在组件样式中应优先使用现有辅助类或直接引用 CSS 变量，避免写死 `#277EFB`、`8px` 等值。

## 5. AI 编码指令 (Styling Rules)

1. **严禁硬编码颜色或尺寸**：所有样式必须通过 CSS 变量或实用类实现。
2. **统一字体**：使用系统字体栈 `PingFang SC, Microsoft YaHei, system-ui`，不引入外部 Google Fonts。
3. **气泡对齐**：聊天气泡布局使用 `flex items-start`，确保头像与气泡顶部基线对齐。
4. **暗色模式适配**：通过 `data-theme="dark"` 自动切换变量，组件无需额外处理。
5. **全局滚动条**：在 `global.scss` 中统一样式，确保在暗色模式下颜色可见。

## 6. 维护与扩展指引

- **新增颜色**：先在 `src/styles/design-tokens.scss` 中补充 token，再回写文档示例。
- **新增实用类**：优先放入 `src/styles/tailwind-overrides.scss` 或 `src/styles/global.scss`，不要创建脱离现有入口的新样式文件。
- **图标扩展**：将 SVG 放入 `src/assets/icons/`，并在 `icon-index.md` 中登记。

## 7. 导航项样式规范 (Sidebar Navigation Item)

所有面板侧边栏的导航项（`.nav-item`）必须统一使用以下样式规范：

### 7.1 统一样式参数

```scss
.nav-item {
  display: flex;
  flex-direction: row;           // 水平排列：图标在前，文字在后
  align-items: center;            // 垂直居中
  gap: var(--dt-spacing-sm);     // 图标与文字间距 8px
  padding: var(--dt-spacing-sm) var(--dt-spacing-md);  // 上下8px 左右12px
  height: 40px;                   // 固定高度 40px
  box-sizing: border-box;        // 盒模型计算
  border-radius: var(--dt-radius-md);  // 圆角 6px
  cursor: pointer;
  transition: all var(--dt-transition-base);
}

/* 图标 */
.nav-icon {
  font-size: var(--dt-icon-size-md, 16px);  // 图标 16px
  flex-shrink: 0;
}

/* 文字 */
.nav-label {
  flex: 1;
  font-size: var(--dt-font-size-base, 14px);  // 文字 14px
  color: var(--dt-text-primary);
}

/* 悬浮态 */
.nav-item:hover {
  background: var(--dt-bg-session-hover);
}

/* 激活态 */
.nav-item.active {
  background: var(--dt-bg-session-active);
  color: var(--dt-brand-color);
  font-weight: var(--dt-font-weight-semibold);
}
```

### 7.2 已统一的面板

以下面板的导航样式已统一为此规范：

| 文件 | 组件 |
| :--- | :--- |
| `ContactsPanel.vue` | 通讯录侧边导航 |
| `TodoPanel.vue` | 待办侧边导航 |
| `DocumentsPanel.vue` | 云文档侧边导航 |
| `WorkbenchPanel.vue` | 工作台侧边导航 |

### 7.3 设计原则

- **固定高度**：`height: 40px` 确保所有导航项高度一致
- **水平布局**：`flex-direction: row` 确保图标在左、文字在右
- **使用 Design Token**：所有颜色、间距、圆角必须使用 `var(--dt-*)` 变量
- **统一激活态**：使用 `var(--dt-bg-session-active)` 背景 + 左侧蓝色指示条
