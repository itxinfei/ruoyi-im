# 21 IM 桌面端页面规范 (DingTalk Desktop Spec)

> **用途**：定义消息模块在 Windows 桌面端的执行标准，确保 UI 逻辑与钉钉保持高度一致。
> **核心原则**：高信息密度、触发式交互、8px 栅格系统。
> **版本**：v2.0 - 实现版（AI 代码生成专用）

---

## 1. 页面总布局 (Three-Pane Layout)

### 1.1 骨架结构 (Master Structure)

```vue
<!-- 根容器 - 严禁修改 -->
<div class="im-root">
  <aside class="im-sidebar">...</aside>
  <aside class="im-session-list">...</aside>
  <main class="im-chat-panel">...</main>
  <aside class="im-detail-drawer">...</aside>
</div>
```

```scss
.im-root {
  display: flex;
  height: 100vh;
  width: 100vw;
  overflow: hidden;  // 严禁出现全局滚动条
}

.im-sidebar {
  width: 64px;              // L1 固定宽度
  min-width: 64px;

  flex-shrink: 0;

  height: 100%;
  display: flex;

```

## 3. 会话列表 (Session List)

### 3.1 结构规范

```vue
<aside class="im-session-list">
  <!-- 搜索栏 -->
  <div class="session-search">
    <el-input placeholder="搜索" prefix-icon="el-icon-search" />
  </div>

  <!-- 会话列表 -->
  <div class="session-scroll">
    <div
      v-for="session in sessions"
      :key="session.id"
      class="session-item"
      :class="{ active: session.id === currentSessionId }"
    >
      <el-avatar :size="40" :src="session.avatar" class="session-avatar" />
      <div class="session-content">
        <div class="session-title">{{ session.name }}</div>
        <div class="session-preview">{{ session.lastMessage }}</div>
      </div>
      <div v-if="session.unread > 0" class="session-badge">
        {{ session.unread > 99 ? '99+' : session.unread }}
      </div>
    </div>
  </div>
</aside>
```

## 4. 聊天主区域 (Chat Panel)

### 4.1 结构规范

```vue
<main class="im-chat-panel">
  <!-- 聊天头部 -->
  <header class="chat-header">
    <div class="chat-title">会话名称</div>
    <div class="chat-actions">
      <i class="el-icon-more"></i>
    </div>
  </header>

  <!-- 消息列表 -->
  <div class="chat-messages" ref="messagesRef">
    <div class="message-list">
      <!-- 消息气泡组件 -->
      <ChatMessageBubble
        v-for="msg in messages"
        :key="msg.id"
        :message="msg"
        :is-own="msg.senderId === currentUserId"
        @mouseenter="onMessageHover(msg.id)"
        @mouseleave="onMessageLeave(msg.id)"
      />
    </div>
  </div>

  <!-- 输入区 -->
  <div class="chat-input-area">
    <ChatInputArea />
  </div>
</main>
```

## 5. 禁止项 (UI Anti-Patterns)

1. **严禁** 使用 `box-shadow` 做大面积投影（钉钉 UI 极其扁平）
2. **严禁** 气泡使用渐变色
3. **严禁** 导航栏使用明亮的彩色背景
4. **严禁** 出现圆形头像（钉钉桌面端基因为方圆 4px）
5. **严禁** 在 L1/L2 区域出现任何形式的 Mobile Tab Bar 或 底部导航
6. **严禁** 使用 `el-row/el-col` 进行核心栅格分配
7. **严禁** 使用超过 `12px` 的圆角
8. **严禁** 使用 `position: absolute` 做主布局
