# 21 IM 桌面端页面规范 (DingTalk Desktop Spec)

> **用途**：定义消息模块在 Windows 桌面端的执行标准，确保 UI 逻辑与钉钉保持高度一致。
> **核心原则**：高信息密度、触发式交互、8px 栅格系统。

---

## 1. 页面总布局 (The Three-Pane Layout)

消息主页面必须采用原生的 Flexbox 结构，严禁响应式折叠，严禁使用 `el-row/col`。

### 1.1 骨架尺寸 (Master Dimensions)
1. **L1 一级主导航 (Sidebar)**: 固定宽度 `64px`（`--dt-nav-sidebar-width`）。亮色模式使用 `--dt-brand-color` (#277EFB)，暗色模式使用 `--dt-nav-sidebar-bg` (#1e293b)。
   - **Logo 区域**: 顶部第一个位置，尺寸 `64x64`（`--dt-logo-size`）。图标居中，圆角 `8px`。
   - **Logo 样式**: 白色文字，字号16px，字重700。
2. **L2 二级会话栏 (Session List)**: 固定宽度 `250px`。背景色 `#FFFFFF`。
3. **L3 三级内容区 (Chat Panel)**: `flex: 1`，最小宽度 `480px`。背景色 `#f5f6f7`（`--dt-bg-chat`）。
4. **L4 详情抽屉 (Detail Drawer)**: 悬浮/滑出式，固定宽度 `320px`。背景色 `#FFFFFF`。

### 1.2 根节点约束 (Viewport Hardening)
- `height: 100vh; overflow: hidden; display: flex;`。
- 仅允许子区域内滚动，全局禁止主滚动条。

---

## 2. 8px 黄金比例尺 (Visual Rhythm)

所有组件开发必须严格遵循 **8px 步进**：
- **间距 (Spacing)**: 4, 8, 12, 16, 24, 32, 48。
- **圆角 (Radius)**: 
  - **4px**: 按钮、输入框、头像、小标签。
  - **8px**: 消息气泡、弹窗容器、卡片。
- **图标 (Icons)**: 
  - **20x20**: 主功能图标。
  - **16x16**: 辅助操作图标。

---

## 3. 二级会话栏规格 (Session List Details)

### 3.1 会话项 (Session Item)
- **高度**: 固定 `56px`（`--dt-session-item-height`）。
- **头像**: `40x40`（`--dt-avatar-session`），圆角 `4px`。
- **未读数 (Badge)**: 
  - 背景色 `#FF4D4F`（`--dt-unread-color`）。
  - 样式：正圆（1-2位数字）或 胶囊形（3位以上），严禁正方形。
- **文字层级**:
  - **标题**: `14px`, `#171A1D` (90%)。
  - **摘要**: `12px`, `rgba(23, 26, 29, 0.4)` (40%)。
- **选中态**: 背景色 `#E6F2FF`（`--dt-bg-session-active`），左侧边缘带一条 `3px` 宽的蓝色指示条 (`#277EFB`)。

---

## 4. 聊天主区域规格 (Chat Panel Details)

### 4.1 消息气泡 (Message Bubble)
- **对齐基线**: 头像顶部与气泡容器顶部对齐。
- **气泡颜色**:
  - **发送方 (Me)**: 背景 `#277EFB`，文字白色。
  - **接收方 (Other)**: 背景 `#FFFFFF`，文字 `#171A1D`。
- **特殊圆角 (DingTalk Style)**:
  - **发送方**: `radius: 8px 4px 8px 8px` (右上角略尖)。
  - **接收方**: `radius: 4px 8px 8px 8px` (左上角略尖)。
- **已读状态**: 仅发送方气泡显示。位于气泡外部左下角，距离气泡边缘 `4px`。显示蓝色 `√` 或 `已读` 字样，字号 `12px`，色值 `rgba(23, 26, 29, 0.4)`（已读）或 `#277EFB`（未读）。

### 4.2 消息间距 (Message Spacing)
- **连续消息 (同一分钟/同一人)**: 间距 `4px`，隐藏头像和昵称。
- **非连续消息**: 间距 `16px`。
- **系统通知 (时间线)**: 居中，`12px`，色值 `rgba(23, 26, 29, 0.4)`。

### 4.3 输入区 (Input Area)
- **工具栏**: 高度 `32px`，图标间距 `16px`。
- **输入框**: `border: none`，Focus 时不产生外边框。
- **发送按钮**: 位于右下角，`32px` 高度，仅在有内容时高亮。

---

## 5. 详情抽屉规范 (Detail Drawer)

- **触发**: 由点击聊天头部右侧 `更多` 图标触发。
- **Section 划分**: 每一块功能使用 `1px` 宽的 `rgba(23, 26, 29, 0.08)` 分割线。
- **开关 (Switch)**: 统一使用 Element Plus 的 `small` 尺寸，激活色为 `#277EFB`。

---

## 6. 禁止项 (UI Anti-Patterns)

1. **严禁** 使用 `box-shadow` 做大面积投影（钉钉 UI 极其扁平）。
2. **严禁** 气泡使用渐变色。
3. **严禁** 导航栏使用明亮的彩色背景。
4. **严禁** 出现圆形头像（钉钉桌面端基因为方圆 4px）。
5. **严禁** 在 L1/L2 区域出现任何形式的 Mobile Tab Bar 或 底部导航。
