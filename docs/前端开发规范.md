## 一、目录结构规范

### 1.1 标准目录结构

```
src/
├── assets/              # 静态资源（图片、字体等）
├── components/          # 全局组件
│   ├── Common/         # 通用组件
│   ├── Chat/            # 聊天相关组件
│   ├── Contacts/        # 通讯录相关组件
│   ├── Workbench/       # 工作台相关组件
│   └── ...              # 其他业务组件
├── composables/         # 组合式函数
├── router/              # 路由配置
├── store/               # Vuex 状态管理
│   └── modules/         # 模块化 store
├── styles/              # 全局样式
├── utils/               # 工具函数
├── views/                # 页面组件
│   ├── auth/            # 认证相关页面
│   ├── admin/           # 后台管理页面
│   ├── MainPage.vue     # 主页面（包含所有模块）
│   └── ...
└── api/                 # API 接口封装
```

### 1.2 组件目录规范

**业务组件按功能模块分组**：
- `Chat/` - 聊天消息相关组件
- `Contacts/` - 通讯录相关组件
- `Workbench/` - 工作台相关组件
- `Documents/` - 云盘文档相关组件
- `Calendar/` - 日历相关组件
- `Todo/` - 待办相关组件
- `Approval/` - 审批相关组件
- `Mail/` - 邮箱相关组件

**通用组件放在 Common/ 目录**：
- 按钮、弹窗、表单等跨业务使用的组件

### 1.3 组件命名规范

**Vue 组件命名**：
- 使用 PascalCase
- 功能明确、语义化
- 例如：`MessageBubble.vue`、`ChatHeader.vue`、`UserDetailDrawer.vue`

**API 文件命名**：
- 使用小写字母加连字符
- 按业务模块分组
- 例如：`im/conversation.js`、`admin.js`、`im/message.js`

## 二、代码编写规范

### 2.1 Vue 组件编写规范

#### 2.1.1 组件结构

```vue
<template>
  <!-- 模板 -->
</template>

<script setup>
// 1. 导入依赖（优先级：Vue API -> 第三方库 -> 本地模块）
import { ref, computed, onMounted } from 'vue'
import { ElMessage } from 'element-plus'

// 2. Props 定义（使用 TypeScript 或 JSDoc 注释）
const props = defineProps({
  modelValue: {
    type: String,
    required: true
  }
})

// 3. Emits 定义
const emit = defineEmits(['update:modelValue', 'change'])

// 4. 响应式数据
const loading = ref(false)
const formData = reactive({})

// 5. 计算属性
const computedValue = computed(() => {
  return props.modelValue.toUpperCase()
})

// 6. 方法定义
const handleSubmit = async () => {
  // 实现逻辑
}

// 7. 生命周期钩子
onMounted(() => {
  // 初始化逻辑
})
</script>

<style scoped lang="scss">
// 组件样式（仅使用 CSS 变量，避免硬编码颜色）
</style>
```

#### 2.1.2 禁止事项

❌ **禁止在模板中使用复杂逻辑**
```vue
<!-- 错误示例 -->
<template>
  <div>{{ list.filter(item => item.active).map(item => item.name).join(', ') }}</div>
</template>
```

✅ **正确做法**
```vue
<template>
  <div>{{ filteredListNames }}</div>
</template>

<script setup>
const filteredListNames = computed(() => {
  return list.filter(item => item.active).map(item => item.name).join(', ')
})
</script>
```

❌ **禁止在模板中直接调用方法**
```vue
<!-- 错误示例 -->
<template>
  <button @click="handleSave()">保存</button>
</template>

<script setup>
const handleSave = () => {
  // 复杂逻辑
}
</script>
```

✅ **正确做法**
```vue
<template>
  <button @click="handleSave">保存</button>
</template>
```

### 2.2 样式编写规范

#### 2.2.1 使用 CSS 变量

✅ **正确** - 使用设计系统变量
```scss
.message-bubble {
  padding: var(--dt-bubble-padding-v) var(--dt-bubble-padding-h);
  background: var(--dt-bubble-left-bg);
  color: var(--dt-text-primary);
  border-radius: var(--dt-bubble-radius);
}
```

❌ **错误** - 硬编码颜色值
```scss
.message-bubble {
  padding: 10px 14px;
  background: #ffffff;
  color: #1f2329;
  border-radius: 10px;
}
```

#### 2.2.2 8pt 栅格体系

使用 4px 的倍数作为间距单位：
- `--dt-spacing-xs: 4px`
- `--dt-spacing-sm: 8px`
- `--dt-spacing-md: 12px`
- `--dt-spacing-lg: 16px`
- `--dt-spacing-xl: 24px`
- `--dt-spacing-2xl: 32px`

#### 2.2.3 字号与行高规范

| 用途 | 字号 | 行高 |
|------|------|------|
| 大标题 | 18px | 1.3 |
| 标题 | 16px | 1.4 |
| 正文 | 14px | 1.6 |
| 辅助文本 | 12px | 1.5 |
| 标签 | 11px | 1.4 |

#### 3.2.1 二级菜单实现方式

**方式一：左侧二级导航（推荐用于数据量大的模块）**
```
┌─────────────────────────────────┐
│ 一级导航 (64px) │ 二级导航 (200px) │ 内容区 │
│                  │ - 全部           │        │
│                  │ - 未读           │        │
│                  │ - 置顶           │        │
│                  │ - 免打扰         │        │
└─────────────────────────────────┘
```

**方式二：Tab 切换（推荐用于数据量小的模块）**
```
┌─────────────────────────────────┐
│ 一级导航 (64px) │ Tab: 全部 | 内容区 │
│                  │ Tab: 未读  │        │
└─────────────────────────────────┘
```

### 7.3 常用工具函数

```javascript
// utils/format.js - 时间格式化
export function formatTime(timestamp) {
  const date = new Date(timestamp)
  const now = new Date()
  // 今天
  if (date.toDateString() === now.toDateString()) {
    return `今天 ${formatHHmm(date)}`
  }
  // 昨天
  const yesterday = new Date(now)
  yesterday.setDate(yesterday.getDate() - 1)
  if (date.toDateString() === yesterday.toDateString()) {
    return `昨天 ${formatHHmm(date)}`
  }
  // 今年
  if (date.getFullYear() === now.getFullYear()) {
    return `${date.getMonth() + 1}月${date.getDate()}日 ${formatHHmm(date)}`
  }
  // 往年
  return `${date.getFullYear()}年${date.getMonth() + 1}月${date.getDate()}日 ${formatHHmm(date)}`
}

function formatHHmm(date) {
  return `${date.getHours().toString().padStart(2, '0')}:${date.getMinutes().toString().padStart(2, '0')}`
}
```
