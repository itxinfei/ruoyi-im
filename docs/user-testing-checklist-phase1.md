## Task 1.1: 消息发送状态 UI

### 发送中状态
- [ ] 显示转圈动画
- [ ] 位置正确（消息气泡右侧）
- [ ] 动画流畅

### 已送达状态
- [ ] 灰色对勾（#999999）
- [ ] 图标清晰

### 已读状态（单聊）
- [ ] 蓝色双对勾
- [ ] 状态正确更新

### 已读状态（群聊）
- [ ] 显示已读人数（如"已读 2/3"）
- [ ] 点击查看已读成员列表
- [ ] 未读成员置灰

### 发送失败状态
- [ ] 红色感叹号
- [ ] 红色背景高亮
- [ ] 显示"点击重发"提示
- [ ] 悬停显示错误原因
