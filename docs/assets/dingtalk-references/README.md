## 📐 截图采集规范

### 软件要求
- 钉钉 PC 端版本 **8.2.x**（截至 2026 年最新稳定版）
- 操作系统：**Windows 10/11**（macOS 略有差异，仅作辅助参考）
- 显示器：1920×1080 物理分辨率（标准 DPI），**禁止**用 4K + 缩放采集
- 浏览器：N/A（钉钉 PC 是 Electron 客户端）

### 采集方法
1. **不要全屏截图**：截局部矩形即可，但保留至少 16px 边距给 UI 解读
2. **保留遮罩层**：弹窗/抽屉类必须含半透明遮罩
3. **多状态对照**：同一组件采集"默认 + hover + active + disabled" 4 态
4. **保留刻度**：建议用 PixelPick / PixelPerfect 工具叠加像素尺
5. **文件格式**：统一 PNG（保留透明度），文件名按本清单严格命名

### 标注要求（推荐用 Snipaste / Skitch / Figma）

每张截图必须叠加以下信息：
- **元素尺寸**：宽 × 高 + 圆角值（红色框 + 数字）
- **颜色取值**：用色块 + 十六进制（吸管工具：Color Cop / WhatColor）
- **间距**：组件间距 / padding（绿色双向箭头）
- **文字**：字号 / 字重 / 颜色

### 命名约定
- 主截图：`XX-描述.png`（XX 为两位数字编号）
- 标注版本：`XX-描述-annotated.png`
- 切片版本：`XX-描述-detail-A.png`（仅特定细节放大）
