## 一、设计概述

### 1.1 设计目标

设计一套**钉钉风格**的企业 IM 系统界面，适用于**桌面浏览器**环境，提供高效、清晰的用户体验。

### 1.2 核心设计原则

| 原则 | 说明 |
|------|------|
| **单一视图** | 详情信息使用弹窗展示，避免侧边栏占用空间 |
| **垂直滚动** | 弹窗内所有信息在一个可滚动视图中 |
| **分区清晰** | 使用卡片和分隔线区分不同内容区域 |
| **渐进展示** | 默认显示少量内容，点击展开查看更多 |

### 1.3 布局结构

```
┌─────┬──────────────────┬─────────────────────────────────────────────┐
│ 60px│      280px       │              flex: 1                        │
│ 导航│   会话列表       │            消息区域                          │
│ 栏 │                  │                                             │
│ 📱 │ ┌──────────────┐ │  产品研发群 (128人)            [群详情] [⋯]│
│ 💬 │ │📢 产品研发群 │ │  ───────────────────────────────────────────│
│ 👥 │ │ 下午好... 🔴5│ │  [Message bubbles...]                       │
│ 📋 │ ├──────────────┤ │  [Input area]                                │
│ 📧 │ │👤 王经理     │ │                                             │
│ 📅 │ └──────────────┘ │                                             │
```
