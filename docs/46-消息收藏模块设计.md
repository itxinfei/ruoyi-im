## 二、后端实现 (JDK 1.8 规范)

### 2.1 收藏持久化逻辑

```java
@Service
@RequiredArgsConstructor
public class ImFavoriteServiceImpl implements ImFavoriteService {

    private final ImFavoriteMapper favoriteMapper;

    /**
     * 收藏消息
     * 遵循 JDK 1.8 规范
     */
    @Transactional(rollbackFor = Exception.class)
    public void addFavorite(FavoriteDTO favoriteDTO) {
        ImFavorite favorite = new ImFavorite();
        BeanUtils.copyProperties(favoriteDTO, favorite);
        favorite.setCreateTime(new Date());
        favoriteMapper.insert(favorite);
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 收藏列表逻辑封装

```javascript
/**
 * 收藏列表逻辑封装 (纯 JS)
 */
import { ref } from 'vue';
import { favoriteApi } from '@/api/favorite';

export function useFavorite() {
  const favorites = ref([]);

  const loadFavorites = async (params) => {
    const response = await favoriteApi.list(params);
    favorites.value = response.list || [];
  };

  const deleteFavorite = async (id) => {
    await favoriteApi.remove(id);
    await loadFavorites();
  };

  return {
    favorites,
    loadFavorites,
    deleteFavorite
  };
}
```
