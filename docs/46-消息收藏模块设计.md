## 二、后端实现 (JDK 1.8 规范)

### 2.1 收藏持久化逻辑

```java
@Service
@RequiredArgsConstructor
public class ImFavoriteServiceImpl implements ImFavoriteService {

    private final ImFavoriteMapper favoriteMapper;

    /**
     * 收藏消息
     * 遵循 JDK 1.8 规范
     */
    @Transactional(rollbackFor = Exception.class)
    public void addFavorite(FavoriteDTO favoriteDTO) {
        ImFavorite favorite = new ImFavorite();
        BeanUtils.copyProperties(favoriteDTO, favorite);
        favorite.setCreateTime(new Date());
        favoriteMapper.insert(favorite);
    }
}
```

## 三、前端实现 (纯 JS 规范)

### 3.1 收藏列表逻辑封装

```javascript
/**
 * 收藏列表逻辑封装 (纯 JS)
 */
import { ref } from 'vue';
import { favoriteApi } from '@/api/favorite';

export function useFavorite() {
  const favorites = ref([]);

  const loadFavorites = async (params) => {
    const response = await favoriteApi.list(params);
    favorites.value = response.list || [];
  };

  const deleteFavorite = async (id) => {
    await favoriteApi.remove(id);
    await loadFavorites();
  };

  return {
    favorites,
    loadFavorites,
    deleteFavorite
  };
}
```

## 四、UI 表现与对齐规范 (DingTalk Style)

### 4.1 收藏列表 (Favorites List)
- **页面布局**：占据应用右侧主内容区（L3）。背景色 `#F2F4F5`。
- **收藏卡片 (Favorite Card)**：
  - 宽度 `100%`，最大宽度限制为 `800px`。
  - 背景色 `#FFFFFF`，圆角 `8px`，外边距（上下间距）`16px`。内边距 `16px`。
- **内容呈现**：
  - 顶部：来源头像（`24x24`）、昵称、时间（`12px`，浅灰色）。
  - 中部：被收藏的具体消息内容（支持文本、图片缩略图、文件卡片）。

### 4.2 交互动作 (Actions)
- **定位按钮**：卡片右下角提供”跳转至聊天”按钮，文字色 `#0089FF`，点击后立即切换到消息模块并高亮原消息。
- **取消收藏**：卡片右上角 `...` 菜单内，提供“取消收藏”选项，文字为红色 `#FF4D4F`。
