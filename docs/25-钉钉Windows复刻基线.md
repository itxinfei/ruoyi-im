# 25 钉钉 Windows 复刻基线

> 用途：按页面定义"必须复刻到什么程度"与"怎么验收"，避免复刻任务只停留在主观判断。
> 说明：本文件不描述代码实现细节，只定义页面级目标与验收口径。

## 1. 适用范围

1. 消息主页面（会话栏 + 聊天区 + 详情区）
2. 搜索页面
3. 群详情/个人详情页面
4. 左侧导航模块页面（通讯录、工作台、云盘、待办等）

## 2. 复刻等级定义

1. `L1`：结构一致。布局和模块层级与钉钉 Windows 一致。
2. `L2`：视觉一致。间距、尺寸、状态色、组件密度一致。
3. `L3`：交互一致。点击、悬停、右键、空状态、失败反馈一致。
4. `L4`：功能闭环。入口真实可达，主流程和异常流程可完成。

交付要求：

1. IM 主链路页面最低达到 `L4`。
2. 非 IM 页面最低达到 `L2`，且关键入口要达到 `L3`。

## 4. 验收口径

1. 每个页面至少提供一组“当前 vs 钉钉”对比截图。
2. 验收必须同时覆盖视觉、交互和功能闭环。
3. 如果某项无法达到对应等级，必须在 `04-待确认问题.md` 写明原因。
4. 验收结果回写到 `03-实现状态矩阵.md` 与任务卡 `acceptance`。

## 6. 对齐与细节指南 (Alignment Guide)

为了确保“钉钉感”，AI 与开发人员必须遵循以下 Windows 桌面端的核心尺寸与对齐策略：

### 6.1 核心骨架尺寸
- **L1 级导航栏 (左侧)**：固定 `72px`。背景色 `#2B2D33`，图标居中对齐，激活态使用白色背景/高亮图标。
- **L2 级会话栏 (中间)**：固定 `250px`。背景色 `#FFFFFF`，右侧带一条 `1px` 的 `#EBF5FF` 边框。
- **L3 级内容区 (右侧)**：`flex: 1` 占据剩余空间。
- **全局间距单位**：采用 `8px` 步进系统（所有 `gap`、`padding` 必须是 8, 16, 24, 32 的倍数，特殊微调允许 4, 12）。

### 6.2 聊天组件微观对齐
- **头像规范**：
  - **尺寸**：统一 `36px * 36px`。
  - **圆角**：固定 `4px`（钉钉桌面端非圆形）。
  - **对齐**：头像顶部与消息气泡顶部水平对齐。
- **消息气泡**：
  - **内边距 (Padding)**：上下 `8px`，左右 `12px`。
  - **气泡间距**：同侧气泡垂直间距 `8px`，不同用户间距 `16px`。
  - **文字行高**：`1.5`（14px 字号对应 21px/22px 行高）。
- **工具栏 (输入框上方)**：
  - **图标尺寸**：`20px`。
  - **图标间距**：水平间距 `16px`。

### 6.3 列表项 (List Item)
- **高度**：标准会话列表项高度为 `64px`。
- **文字层级**：
  - 标题：`14px`，`#171A1D` (90%)。
  - 摘要：`12px`，`rgba(23, 26, 29, 0.4)`。
- **选中态**：背景色使用 `var(--dt-bg-selected)` (#EBF5FF)，左侧增加一条 `3px` 宽的蓝色指示条。
