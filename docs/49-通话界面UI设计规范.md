# 49 通话界面UI设计规范

> **用途**：定义音视频通话界面的完整UI交互规范
> **更新时间**：2026-04-01
> **优先级**：P2
> **对标产品**：钉钉 Windows 客户端 8.2.0

---

## 一、设计概述

### 1.1 通话场景

| 场景 | 说明 | 优先级 |
|-----|------|-------|
| **单聊语音通话** | 两人之间语音通话 | P1 |
| **单聊视频通话** | 两人之间视频通话 | P1 |
| **群组语音通话** | 多人语音通话（最多30人） | P2 |
| **群组视频通话** | 多人视频通话（最多9人） | P2 |

### 1.2 通话状态流转

```
[IDLE] → [CALLING] → [RINGING] → [CONNECTED] → [ENDED]
   ↓         ↓           ↓            ↓
发起中   呼叫中      响铃中      通话中
   ↓         ↓           ↓            ↓
   └─────── [REJECTED] ──────────────┘
            已拒绝
```

### 1.3 通话规则 (C-009)

- **短时断线**（< 3s）：自动重连，UI显示"正在重新连接..."
- **长时断线**（≥ 3s）：提示用户手动重拨，显示"连接已断开，是否重拨？"
- **群通话规模**：视频最多9人同时在线，语音最多30人
- **通话记录保留**：90天

## 二、来电界面 (Incoming Call Interface)

### 2.1 全屏来电弹窗

- **尺寸**：宽度 `480px`，高度 `560px`
- **位置**：屏幕中央，水平垂直居中
- **背景**：渐变色 `rgba(0, 0, 0, 0.85)` → `rgba(43, 45, 51, 0.95)`
- **圆角**：`16px`
- **阴影**：`0 20px 60px rgba(0, 0, 0, 0.3)`
- **遮罩**：全屏半透明遮罩 `rgba(0, 0, 0, 0.5)`

### 2.2 来电头部区域

- **高度**：`240px`
- **内容**：
  - 发起者头像：`120x120`，圆角 `60px`（圆形）
  - 发起者昵称：`20px`，粗体，白色 `#FFFFFF`
  - 通话类型：`14px`，`rgba(255, 255, 255, 0.8)`
  - 通话时长倒计时：`32px`，白色，从50s倒计时

### 2.3 来电操作区域

- **高度**：`200px`
- **布局**：水平居中，两个大按钮
- **按钮尺寸**：`72x72`
- **按钮圆角**：`50%`（正圆）
- **按钮间距**：`80px`

#### 2.3.1 接听按钮 (Accept)

- **背景**：渐变色 `#52C41A` → `#3BB636`
- **图标**：电话图标 `36x36`，白色
- **文字**：接听（`14px`，白色）
- **Hover**：放大 `1.05` 倍，阴影 `0 4px 12px rgba(82, 196, 26, 0.4)`

#### 2.3.2 拒绝按钮 (Reject)

- **背景**：渐变色 `#FF4D4F` → `#F5222D`
- **图标**：挂断图标 `36x36`，白色
- **文字**：拒绝（`14px`，白色）
- **Hover**：放大 `1.05` 倍，阴影 `0 4px 12px rgba(255, 77, 79, 0.4)`

### 2.4 动画效果

- **出现动画**：
  - 淡入 + 缩放
  - 时长：`300ms`
  - 缓动函数：`cubic-bezier(0.34, 1.56, 0.64, 1)`（弹性效果）

- **按钮点击动画**：
  - 按下时缩小 `0.9` 倍
  - 抬起时恢复 `1.0` 倍
  - 时长：`150ms`

## 三、通话中界面 (In-Call Interface)

### 3.1 全屏通话窗口

- **尺寸**：全屏 `100vw x 100vh`
- **背景**：黑色 `#000000`
- **遮罩**：无遮罩（独占全屏）

### 3.2 视频通话布局

#### 3.2.1 主视频区域（对方）

- **位置**：全屏填充
- **尺寸**：`100% x 100%`
- **对象适配**：`object-fit: cover`
- **背景**：深灰色 `#1F2937`

#### 3.2.2 自己的小窗口（画中画）

- **尺寸**：`200x280`
- **位置**：右下角，距离底部 `40px`，距离右侧 `40px`
- **圆角**：`12px`
- **边框**：`2px solid rgba(255, 255, 255, 0.2)`
- **拖拽**：支持鼠标拖动到屏幕任意位置
- **双击**：双击切换大小（大窗口 ↔ 小窗口）

#### 3.2.3 顶部信息栏

- **位置**：顶部居中
- **高度**：`60px`
- **背景**：线性渐变 `rgba(0, 0, 0, 0.6)` → `transparent`
- **内容**：
  - 通话时长：`18px`，白色
  - 网络状态：图标 + 文字（`#52C41A` 优 / `#FA8C16` 良 / `#FF4D4F` 差）
  - 通话类型：`14px`，`rgba(255, 255, 255, 0.8)`

#### 3.2.4 底部控制栏

- **位置**：底部居中
- **高度**：`100px`
- **背景**：线性渐变 `transparent` → `rgba(0, 0, 0, 0.6)`
- **内容**：控制按钮组，间距 `24px`

- **图标颜色**：白色 `#FFFFFF`
- **Hover**：背景 `rgba(255, 255, 255, 0.25)`，放大 `1.05` 倍

### 3.3 语音通话布局

#### 3.3.1 中央头像区域

- **位置**：屏幕中央
- **尺寸**：`240x240`
- **圆角**：`120px`（圆形）
- **内容**：
  - 发起者头像：`240x240`
  - 背景渐变：`#277EFB` → `#0656C6`
  - 波纹动画：中心向外扩散的音频波形

#### 3.3.2 波纹动画（语音可视化）

- **位置**：头像周围
- **层数**：3层
- **动画**：
  - 从中心向外扩散
  - 时长：`2s`
  - 无限循环
  - 透明度：`0.6` → `0`
  - 尺寸：`240px` → `360px`

#### 3.3.3 信息显示

- **位置**：头像下方
- **内容**：
  - 发起者昵称：`24px`，粗体，白色
  - 通话时长：`20px`，`rgba(255, 255, 255, 0.8)`
  - 间距：`16px`

#### 3.3.4 底部控制栏

与视频通话相同，但**不显示**：
- 摄像头按钮
- 切换摄像头按钮
- 屏幕共享按钮

**显示**：
- 静音
- 扬声器
- 聊天
- 成员（群组通话）
- 最小化
- 挂断

## 四、多人通话界面 (Group Call Interface)

### 4.1 视频多人通话布局

#### 4.1.1 网格布局规则

- **2-4人**：`2x2` 网格
- **5-9人**：`3x3` 网格
- **单元格尺寸**：自适应，等比例分配
- **单元格间距**：`8px`
- **单元格圆角**：`8px`

#### 4.1.2 单元格内容

每个视频单元格包含：

| 元素 | 位置 | 尺寸 | 说明 |
|-----|------|------|------|
| **视频区域** | 填充 | `100%` | 视频流，`object-fit: cover` |
| **用户信息** | 左上角 | - | 头像 `32x32` + 昵称 `14px` |
| **静音图标** | 左下角 | `24x24` | 红色，仅静音时显示 |
| **网络状态** | 右上角 | `16x16` | 绿/橙/红，网络质量差时显示 |
| **说话标识** | 底部居中 | `40x4` | 绿色条，说话时显示 |

#### 4.1.3 自己的窗口

- **位置**：右下角固定（画中画）
- **尺寸**：`200x280`
- **可拖拽**：鼠标拖动
- **可切换**：双击切换主视频窗口

#### 4.1.4 主讲人高亮

- **主讲人标识**：黄色边框 `3px solid #FA8C16`
- **主讲人标签**：黄色标签 `主讲人`，`12px`
- **位置**：视频左上角，用户信息旁

### 4.2 语音多人通话布局

#### 4.2.1 圆形布局

- **中心**：自己的头像 `120x120`
- **周围**：其他成员头像环绕，半径 `200px`
- **头像尺寸**：`80x80`
- **头像间距**：均匀分布
- **最大人数**：30人

#### 4.2.2 发言高亮

- **发言者**：头像放大 `1.2` 倍，黄色边框 `3px solid #FA8C16`
- **波形动画**：头像周围显示音频波形
- **持续时间**：发言结束后 `1s` 恢复

#### 4.2.3 成员列表

- **位置**：右侧抽屉
- **宽度**：`320px`
- **背景**：`rgba(0, 0, 0, 0.8)`
- **列表项**：
  - 高度：`64px`
  - 内容：头像 `40x40` + 昵称 `14px` + 静音状态
  - Hover：背景 `rgba(255, 255, 255, 0.1)`

## 七、异常处理UI (Error Handling)

### 7.1 网络异常

- **提示**：顶部红色横幅，高度 `48px`
- **内容**：网络不稳定，正在重新连接...
- **图标**：红色感叹号 `20x20`
- **动画**：左右摇晃动画（`500ms`）
- **自动消失**：连接成功后 `1s` 消失

### 7.2 权限拒绝

- **提示**：全屏弹窗
- **内容**：
  - 标题：无法访问麦克风/摄像头
  - 说明：请在系统设置中授予权限
  - 按钮：[前往设置] [取消]
- **尺寸**：宽度 `400px`，高度 `200px`

### 7.3 对方不在线

- **提示**：顶部红色横幅
- **内容**：对方当前不在线，已留言
- **图标**：红色叉号 `20x20`
- **持续时间**：`3s`

### 7.4 通话超时（50s无人接听）

- **提示**：顶部橙色横幅
- **内容**：对方未接听，已留言
- **图标**：橙色时钟 `20x20`
- **持续时间**：`3s`

## 八、动画规范 (Animation Standards)

### 8.1 通用动画时长

| 动画类型 | 时长 | 缓动函数 | 说明 |
|---------|------|---------|------|
| **淡入淡出** | `200ms` | `ease-out` | 界面切换 |
| **滑动** | `250ms` | `cubic-bezier(0.4, 0, 0.2, 1)` | 抽屉/菜单 |
| **缩放** | `300ms` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 弹窗出现 |
| **按钮点击** | `150ms` | `ease-in-out` | 按钮反馈 |
| **波纹扩散** | `2s` | `linear` | 音频波形 |
| **摇晃** | `500ms` | `ease-in-out` | 错误提示 |

### 8.2 语音波形动画

```css
@keyframes voice-wave {
  0% {
    transform: scale(1);
    opacity: 0.6;
  }
  100% {
    transform: scale(1.5);
    opacity: 0;
  }
}

.voice-wave {
  animation: voice-wave 2s linear infinite;
}
```

## 九、响应式设计 (Responsive Design)

### 9.1 最小窗口尺寸

- **最小宽度**：`800px`
- **最小高度**：`600px`
- **低于最小尺寸**：显示提示"窗口太小，请调整窗口大小"

### 9.2 不同屏幕适配

| 屏幕类型 | 分辨率 | 布局调整 |
|---------|-------|---------|
| **桌面** | ≥ 1920x1080 | 全屏显示，最大视频质量 |
| **笔记本** | 1366x768 | 自适应，中等视频质量 |
| **平板** | 768x1024 | 竖屏模式，优化布局 |
| **移动** | ≤ 480x800 | 专用移动端界面 |
