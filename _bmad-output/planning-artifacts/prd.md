# 产品需求文档 - RuoYi-IM 工作台模块

**Author:** Itxinfei
**Date:** 2026-02-28
**版本:** V2.0（工作台聚焦版）

---

## 产品愿景

**核心目标**：私有化部署的企业IM，聚焦工作台体验优化

**产品定位**：
- 轻量化：不做大而全，聚焦核心功能
- 私有化：数据完全自主可控
- 易用：与钉钉对齐，降低学习成本

---

## 工作台模块（核心重点）

### 工作台架构

```
┌─────────────────────────────────────────┐
│              工作台首页                   │
├─────────────────────────────────────────┤
│ ┌─────────┐ ┌─────────────────────────┐ │
│ │ 快捷入口 │ │     通知中心            │ │
│ │         │ │                         │ │
│ │  [图标]  │ │   [未读消息聚合]       │ │
│ │  [图标]  │ │                         │ │
│ │  [图标]  │ └─────────────────────────┘ │
│ │         │ ┌─────────────────────────┐ │
│ └─────────┘ │     待办事项            │ │
│ ┌─────────┐ │                         │ │
│ │ 搜索入口 │ │   [最近3条待办]        │ │
│ └─────────┘ └─────────────────────────┘ │
└─────────────────────────────────────────┘
```

### 功能清单

| 功能 | 优先级 | 说明 |
|-----|--------|------|
| **快捷入口** | P0 | 可配置的应用入口，支持自定义排序 |
| **通知中心** | P0 | 未读消息聚合，点击跳转对应会话 |
| **待办事项** | P0 | 最近3条任务提醒 |
| **搜索入口** | P1 | 全局搜索消息/联系人 |
| **日程概览** | P1 | 今日日程显示 |

### 详细功能需求

#### FR-WT-01: 快捷入口

**功能描述**：
- 工作台顶部显示常用功能入口
- 默认显示：消息、通讯录、文件、日程
- 用户可自定义添加/删除/排序

**交互设计**：
- 入口以图标+文字形式展示
- 支持拖拽排序
- 长按或点击编辑按钮进入编辑模式
- 点击直接跳转对应功能

**数据存储**：
- 用户配置存储在用户设置表中
- 支持配置同步到多端

---

#### FR-WT-02: 通知中心

**功能描述**：
- 聚合显示所有未读消息
- 包括：单聊未读、群聊未读、系统通知
- 显示未读总数红点

**交互设计**：
- 点击展开通知列表
- 每条显示：发送人、摘要、时间
- 点击跳转到对应会话
- 一键已读功能

---

#### FR-WT-03: 待办事项

**功能描述**：
- 显示最近3条待办任务
- 包括：创建时间、截止时间、任务内容
- 支持快速处理

**交互设计**：
- 显示任务标题和截止时间
- 颜色标识紧急程度（红色：今日到期，灰色：已逾期）
- 点击查看详情
- 支持标记完成

---

#### FR-WT-04: 搜索入口

**功能描述**：
- 提供统一的搜索入口
- 支持搜索：消息、联系人、群聊、文件

**交互设计**：
- 点击搜索框进入搜索页面
- 支持关键词匹配
- 显示最近搜索历史
- 搜索结果分类展示

---

#### FR-WT-05: 日程概览

**功能描述**：
- 显示当日日程安排
- 显示开始时间、会议主题、参会人数

**交互设计**：
- 最多显示3条今日日程
- 点击查看完整日程
- 支持快速创建新日程

## 非功能需求

### 性能

- 工作台首屏加载 ≤1秒
- 消息延迟 ≤300ms

### 安全

- 私有化部署，数据不出内网
- 传输加密（WSS）
- 存储加密

### 兼容性

- Chrome / Edge / Firefox / Safari 最新版
- 1920x1080 及以上分辨率
