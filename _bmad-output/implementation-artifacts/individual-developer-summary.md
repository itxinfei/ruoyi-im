## ⚠️ 已移除的内容

为了适合个人开发，以下配置已移除：

| 移除项 | 原因 | 替代方案 |
|--------|------|---------|
| Husky + lint-staged | 过于复杂 | 手动运行 lint |
| Commitlint | 提交规范可选 | 自由提交 |
| Stylelint 复杂配置 | 简化配置 | 基础检查即可 |
| Vitest 测试框架 | 个人项目非必需 | 手动测试 |
| GitHub Actions | 无需 CI/CD | 手动部署 |
| 测试文件 | 减少复杂度 | 手动验证 |
