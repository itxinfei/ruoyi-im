### 一、整体布局架构 (Layout Architecture)

#### 1.1 钉钉管理后台三栏布局

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        钉钉 OA 管理后台布局                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│  ┌──────┐  ┌─────────────┐  ┌──────────────────────────────────────────┐   │
│  │      │  │             │  │                                          │   │
│  │ 左侧 │  │  中间导航区  │  │           右侧内容区                   │   │
│  │ 导航 │  │             │  │                                          │   │
│  │ 栏   │  │  二级菜单    │  │     表单 / 数据表格 / 详情             │   │
│  │      │  │     列表     │  │                                          │   │
│  │ 可折叠 │  │   可配置     │  │           自适应宽度                   │   │
│  │      │  │             │  │                                          │   │
│  └──────┘  └─────────────┘  └──────────────────────────────────────────┘   │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

**关键属性**:
- **左侧主导航**: 可折叠宽度（展开 200-240px，折叠 48-64px），图标式垂直导航
- **中间导航区**: 二级菜单/功能列表，支持分组和标题分隔
- **右侧内容区**: 自适应宽度，显示表单、数据表格或详情页面
- **顶部 Header**: 包含面包屑导航、用户信息、通知、搜索

### 二、色彩系统 (Color System)

#### 2.1 钉钉品牌色值

根据研究结果，钉钉的色彩系统特征：

| 色彩类型 | 色值 (参考) | 应用场景 |
|---------|------------|---------|
| **品牌主色** | 蓝色渐变 (#0089FF → #0066CC) | 主要按钮、链接、选中状态 |
| **辅助色** | 紫色系 (2025新增) | 高级功能、AI特性 |
| **成功色** | #00C853 | 成功提示、完成状态 |
| **警告色** | #FFAB00 | 警告提示、待处理 |
| **错误色** | #FF3D00 | 错误提示、危险操作 |
| **背景色** | #F5F7FA | 页面背景 |
| **卡片背景** | #FFFFFF | 内容卡片背景 |
| **边框色** | #DCDFE6 | 分割线、边框 |
| **文本主色** | #303133 | 主要文字 |
| **文本辅色** | #606266 | 次要文字 |
| **文本禁用** | #C0C4CC | 禁用状态文字 |

#### 2.2 暗色模式支持

- 所有钉钉定义的色彩规范均适配 DarkMode 样式
- 确保文字与背景有高对比度（黑底白字 或 白底黑字）
- 自定义颜色建议支持 DarkMode 以保持一致的用户体验

### 三、组件设计规范 (Component Design)

#### 3.1 导航栏组件

**左侧主导航栏**:
- 展开状态: 200-240px 宽度
- 折叠状态: 48-64px 宽度（仅显示图标）
- 支持分组标题和分割线
- 选中项左侧有蓝色指示条
- 悬停显示功能名称 tooltip

**中间导航区**:
- 清晰的层级缩进（一级 16px，二级 32px）
- 展开/收起动画平滑过渡（0.15s）
- 子项圆点指示器
- 支持图标 + 文字组合

#### 3.2 表单组件

| 组件 | 设计规范 |
|------|---------|
| **输入框** | 高度 36px，边框 1px，圆角 4px，内边距 8px 12px |
| **按钮** | 高度 32px（主按钮）、36px（次按钮），圆角 4px |
| **选择器** | 下拉选项最大高度 256px，带搜索功能 |
| **日期选择** | 与输入框一致的高度和样式 |
| **开关** | 宽度 44px，高度 22px |
| **单选/复选** | 尺寸 16px，间距 8px |

#### 3.3 数据表格

- **表头**: 背景色 #F5F7FA，文字颜色 #606266，高度 48px
- **行高**: 56px（数据行），48px（小尺寸行）
- **边框**: 1px 实线 #EBEEF5
- **斑马纹**: 偶数行背景色 #FAFAFA
- **悬停**: 背景色 #F5F7FA
- **选中**: 背景色 #ECF5FF

## 技术实施要点

### 前端技术栈确认

- **框架**: Vue 3 (Composition API)
- **UI 组件**: Element Plus
- **状态管理**: Vuex / Pinia
- **路由**: Vue Router
- **HTTP**: Axios
- **样式**: SCSS + CSS Variables

### 关键技术决策

| 决策点 | 选择方案 | 理由 |
|--------|---------|------|
| **主题方案** | CSS Variables | 动态切换、暗色模式支持 |
| **布局方案** | 三栏布局（仿钉钉） | 符合目标风格 |
| **组件库** | Element Plus + 自定义 | 既有基础 + 定制化 |
| **状态管理** | 保持现有 Vuex | 减少改动成本 |
| **图表库** | ECharts | 功能强大、中文友好 |
