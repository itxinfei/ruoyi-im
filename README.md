# RuoYi-IM 即时通讯与协同平台

> **定位**：面向企业内网场景的私有化即时通讯平台，产品形态深度复刻 **钉钉 Windows 8.2.0**。
> **架构**：Java 1.8 + Vue 3 (Composition API) + WebSocket/Netty + MySQL + Redis。

## 🤖 大模型接入协议 (LLM Protocol)

本项目采用 **AI-Native** 研发模式，AI 代理必须严格遵守以下执行链：

1.  **强制阅读**：进入项目后必须先阅读 [05-AI执行文档使用协议.md](docs/05-AI执行文档使用协议.md)。
2.  **任务驱动**：所有修改必须基于 [llm-tasks.yaml](docs/llm-tasks.yaml) 中的任务卡，严禁进行范围外重构。
3.  **回写闭环**：任务完成后，必须同步更新 [03-实现状态矩阵.md](docs/03-实现状态矩阵.md) 和 [12-需求点清单.md](docs/12-需求点清单.md)。
